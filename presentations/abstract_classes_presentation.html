🤔 מה זה מחלקה אבסטרקטית?

מחלקה אבסטרקטית היא כמו תבנית או מדריך למחלקות אחרות. זה כמו מתכון בישול שמגדיר את השלבים הכלליים, אבל כל שף יכול להכין אותו בצורה שונה!

🔧 איך כותבים מחלקה אבסטרקטית?

בפייתון אנחנו משתמשים במודול abc (Abstract Base Classes):

from abc import ABC, abstractmethod

class Character(ABC):  # יורשת מ-ABC
    def __init__(self, name, health):
        self.name = name
        self.health = health
    
    # מתודה רגילה - כל דמות יכולה להשתמש בה
    def introduce(self):
        return f"שלום! קוראים לי {self.name}"
    
    # מתודה אבסטרקטית - חייבת להיות ממומשת!
    @abstractmethod
    def attack(self):
        pass  # כל דמות חייבת להגדיר איך היא תוקפת

🔑 נקודות חשובות:

  • @abstractmethod - מסמן שהמתודה חייבת להיות ממומשת
  • ABC - האב של כל המחלקות האבסטרקטיות
  • pass - אומר "אין קוד כאן, רק הגדרה"

⚔️ דוגמה מלאה: דמויות במשחק

from abc import ABC, abstractmethod

class Character(ABC):
    def __init__(self, name, health):
        self.name = name
        self.health = health
    
    @abstractmethod
    def attack(self):
        pass
    
    @abstractmethod  
    def special_power(self):
        pass

# יישום קונקרטי - קוסם
class Wizard(Character):
    def attack(self):
        return f"{self.name} יורה כדור אש! 🔥"
    
    def special_power(self):
        return f"{self.name} משתמש בקסם הגנה! ✨"

# יישום קונקרטי - קשת
class Archer(Character):
    def attack(self):
        return f"{self.name} יורה חץ חד! 🏹"
    
    def special_power(self):
        return f"{self.name} יורה חץ מתפוצץ! 💥"

🎮 שימוש במשחק:

# יצירת דמויות
gandalf = Wizard("גנדלף", 100)
legolas = Archer("לגולס", 80)

# כל דמות תוקפת בצורה שלה
print(gandalf.attack())     # גנדלף יורה כדור אש! 🔥
print(legolas.attack())     # לגולס יורה חץ חד! 🏹

❌ מה קורה אם שוכחים למימש מתודה?

class BrokenWarrior(Character):
    def __init__(self, name, health):
        super().__init__(name, health)
    
    # אופס! שכחנו למימש את attack() ו-special_power()!
    pass

# ננסה ליצור לוחם...
warrior = BrokenWarrior("ארגון", 120)
# TypeError: Can't instantiate abstract class BrokenWarrior with abstract methods attack, special_power

🌟 שימושים נפוצים בעולם האמיתי

1. מערכות תשלום 💳

class PaymentProcessor(ABC):
    @abstractmethod
    def process_payment(self, amount):
        pass

class CreditCard(PaymentProcessor):
    def process_payment(self, amount):
        return f"חיוב כרטיס אשראי: {amount} ₪"

class PayPal(PaymentProcessor):
    def process_payment(self, amount):
        return f"תשלום דרך PayPal: {amount} ₪"

2. צורות גיאומטריות 📐

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass
    
    @abstractmethod
    def perimeter(self):
        pass

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    
    def area(self):
        return 3.14 * self.radius ** 2

🚀 טיפים מתקדמים

1. תכונות (Properties) אבסטרקטיות:

class Animal(ABC):
    @property
    @abstractmethod
    def sound(self):
        pass

class Dog(Animal):
    @property
    def sound(self):
        return "הב הב!"

2. שילוב עם מתודות רגילות:

class Vehicle(ABC):
    def __init__(self, brand):
        self.brand = brand
        self.is_running = False
    
    def start_engine(self):  # מתודה רגילה
        self.is_running = True
        return "המנוע התחיל!"
    
    @abstractmethod
    def move(self):  # מתודה אבסטרקטית
        pass
