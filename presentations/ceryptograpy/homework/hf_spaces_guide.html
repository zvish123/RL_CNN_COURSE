ה-Space צריך בדיוק שני קבצים. לא פחות, לא יותר:

📁 crypto-lab/
├── app.py ← קובץ Python הראשי (חייב להיקרא app.py!)
└── requirements.txt ← רשימת הספריות
⚠️

שם חובה: הקובץ חייב להיקרא app.py בדיוק. קובץ בשם main.py או my_app.py לא יעבוד!

📄 app.py – גרסה מוכנה להעלאה
import gradio as gr

# ─── פונקציות ההצפנה שלכם ─────────────────────
def caesar_cipher(text: str, shift: int, mode: str) -> str:
    if mode == "פענח":
        shift = -shift
    result = []
    for ch in text:
        if ch.isalpha():
            base = ord('A') if ch.isupper() else ord('a')
            result.append(chr(((ord(ch) - base + shift) % 26) + base))
        else:
            result.append(ch)
    return ''.join(result)

# ─── CSS – RTL + תיקון Slider ──────────────────
css = """

"""

# ─── ממשק Gradio ───────────────────────────────
with gr.Blocks(title="מעבדת הצפנים", css=css) as app:
    gr.Markdown("# 🔐 מעבדת הצפנים")
    # ... שאר הממשק שלכם ...

# ✅ ב-Hugging Face: ללא share=True, ללא server_name
app.launch(theme=gr.themes.Soft())
⚠️

הסירו מ-launch(): את share=True – לא נדרש כאן. את server_name="0.0.0.0" – Hugging Face מגדיר זאת בעצמו.

שגיאות נפוצות ופתרונות

!
איך לקרוא את ה-Logs ולפתור בעיות
▼

כשיש שגיאה, הולכים ל-Logs בדף ה-Space ומחפשים שורה אדומה עם המילה Error.

השגיאה ב-Logs הסיבה הפתרון
ModuleNotFoundError: No module named 'X' ספרייה חסרה הוסיפו X ל-requirements.txt
FileNotFoundError: app.py not found הקובץ לא נקרא app.py שנו את שם הקובץ ל-app.py בדיוק
SyntaxError: invalid syntax שגיאת כתיבה בקוד הריצו python app.py מקומית לפני העלאה
AttributeError: 'Blocks' has no attribute 'theme' שימוש ב-theme ב-Blocks במקום launch() העבירו ל-app.launch(theme=...)
Space is sleeping ה-Space לא פעיל (חוסר שימוש) לחצו על Restart this Space
Build took too long ספריות כבדות מדי ודאו ש-requirements.txt מכיל רק מה שצריך
🛠

טיפ: לפני כל העלאה, הריצו את app.py מקומית ווידאו שעובד. כך חוסכים זמן Build מיותר.
