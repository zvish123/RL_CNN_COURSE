פרויקט XO: השדרוג הגדול

ברוכים הבאים למצגת הסיכום של שדרוג מערכת ה-XO AI.

במהלך הלמידה נראה כיצד הפכנו את הקוד מ-"סוכן אקראי" ל-"מעבדת ניסויים חכמה".

מה נלמד היום?

  • הטמעת Heuristics (לוגיקת משחק מובנית).
  • שימוש ב-ExperimentConfig לאוטומציה של טסטים.
  • הפרדה בין נתוני ניסוי ללוגיקה של הקוד.
לוגיקה אסטרטגית #1

זיהוי ניצחון וחסימת יריב

במקום לחכות שה-AI ילמד מטעויות, הוספנו פונקציות שבודקות את הלוח צעד אחד קדימה.

def check_winning_move(self, player): # סריקת כל המשבצות הריקות ובדיקה אם הן מובילות לניצחון for row, col in self.get_available_positions(): self.board[row][col] = player if self.check_winner() == player: self.board[row][col] = " " # ניקוי return (row, col) self.board[row][col] = " " return None def check_blocking_move(self, opponent): # חסימה היא פשוט זיהוי המהלך המנצח של היריב! return self.check_winning_move(opponent)

סדר עדיפויות (Hierarchy)

ה-AI פועל כעת לפי היררכיית החלטות קבועה מראש:

def make_heuristic_move(self): # 1. ניצחון מיידי move = self.check_winning_move(self.current_player) if move: return self.make_move(*move) # 2. חסימת יריב move = self.check_blocking_move(opponent) if move: return self.make_move(*move) # 3. תפיסת המרכז (אסטרטגיה קלאסית) if self.board[1][1] == " ": return self.make_move(1, 1)

ניהול ניסויים עם ExperimentConfig

כדי להשוות בין סוגי AI שונים, יצרנו אובייקט תצורה שמרכז את כל הפרמטרים במקום אחד.

class ExperimentConfig: def __init__(self, name, num_games, gamma, alpha=0.1): self.name = name # שם הניסוי self.num_games = num_games # כמות משחקים להרצה self.gamma = gamma # מקדם דעיכה (Discount Factor) self.alpha = alpha # קצב למידה (Learning Rate) # הגדרת סדרת טסטים להרצה אוטומטית batch = [ ExperimentConfig("Slow_Learner", 1000, 0.5), ExperimentConfig("Fast_Learner", 1000, 0.9) ]

הרצת טסטים מרובים (Batch)

הקוד כעת מסוגל להריץ עשרות ניסויים ברצף, לאסוף את אחוזי הניצחון ולשמור אותם להשוואה.

def run_experiments(configs): for config in configs: # אתחול טורניר עם הגדרות ספציפיות tournament = XOTournament(config.num_games) # הרצה עם הפרמטרים מהקונפיג tournament.train(gamma=config.gamma, alpha=config.alpha) # שמירת תוצאות הניסוי save_results(config.name, tournament.get_stats())
