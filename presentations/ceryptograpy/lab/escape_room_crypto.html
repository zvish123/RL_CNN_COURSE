🔒 משימה 1: זיהוי כתובת השרת

כדי לחדור למערכת, עלינו קודם כל לזהות את כתובת ה-IP המוסתרת של שרת הניהול מתוך קובץ הלוג המבולגן של אבטחת המידע. המערכת פלטה מספר שורות לוג עם כתובות שונות, אך רק אחת מהן מתאימה בדיוק למבנה של הביטוי הרגולרי שהוגדר במערכת הניטור. עליכם למצוא אותה ולהבין מה הפלט שיודפס!

כתב החידה (Python):

import re

log_data = """
INFO 2026-06-08 09:12:01 Connection check from IP:10.0.0.1 port:443
WARN 2026-06-08 09:14:22 Dropped packet from IP:192.168.1.105 port:22
ERROR 2026-06-08 09:15:45 Unauthorized access IP:192.168.43.12 port:8080
DEBUG 2026-06-08 09:17:10 Heartbeat response IP:172.16.254.1 port:9000
"""

pattern = r"IP:(\d{3}\.\d{3})\.(\d{2}\.\d{2})"

match = re.search(pattern, log_data)
if match:
    secret_output = match.group(2).replace(".", "-")
    print(secret_output)
רמז לתלמידים: שימו לב שיש כאן שתי קבוצות לכידה (Groups) בסוגריים. הקבוצה הראשונה match.group(1) לוכדת את החלק הראשון, והקבוצה השנייה match.group(2) לוכדת את החלק שאחרי הנקודה האמצעית. בדקו איזו כתובת בכלל מתאימה למבנה, ומה בדיוק נמצא בתוך הקבוצה השנייה!

כתב החידה (Python):

# secret_cipher.py
def encrypt_transposition(message, key):
    words = message.split(" ")
    processed_words = []

    for w in words:
        if len(w) % 2 == 0:
            processed_words.append(w[::-1])
        else:
            processed_words.append(w)

    message = " ".join(processed_words)
    cols = key
    # calculate the number of matrix rows
    rows = len(message) // key + 1 if len(message) % key != 0 else len(message) // key
    # make the message at (cols X rows) size
    message = message.ljust(cols * rows, " ")
    # create two dimensions array. Inserting the message line by line
    matrix = string_to_array(message, rows, cols)
    # create the cipher text by aggregating letters vertically for the array
    cipher_text = array_string_transposition(matrix)
    return cipher_text

סיווג סודי: מדריך הצפנה אסימטרית

מהי הצפנה אסימטרית? (הקדמה)

בניגוד להצפנה סימטרית (שבה לאליס ובוב יש את אותו המפתח), בהצפנה אסימטרית לכל משתמש יש זוג מפתחות:

  • מפתח ציבורי (Public Key): כמו "מנעול פתוח" שכולם יכולים לראות ולהשתמש בו כדי לנעול (להצפין) הודעות עבורכם.
  • מפתח פרטי (Private Key): המפתח הפיזי שנמצא רק אצלכם בכיס ורק הוא יכול לפתוח (לפענח) את המנעול.

נוסחאות RSA

כדי להצפין ולפענח, RSA משתמשת באריתמטיקה מודולרית:

הצפנה: c = me mod n
פענוח: m = cd mod n

כאשר m הוא המסר, e המפתח הציבורי, d הפרטי, ו-n המודולוס.

תהליך יצירת המפתחות

  1. בוחרים p, q ראשוניים ומחשבים n = p * q.
  2. מחשבים את פונקציית אוילר: Φ(n) = (p-1)*(q-1).
  3. בוחרים e (בדרך כלל מספר ראשוני קטן).
  4. השלב הקריטי: מוצאים d כך שיתקיים:
    (e * d) mod Φ(n) = 1
