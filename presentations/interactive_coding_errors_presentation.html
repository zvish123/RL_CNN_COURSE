קטגוריה 1: שגיאות תחביר (Syntax Errors)

מה זה שגיאות תחביר?

שגיאות בכתיבת הקוד עצמו - דברים שמונעים מהקוד לרוץ בכלל

סוגי שגיאות תחביר נפוצות:

  • Convention שגוי: שמות מחלקות, משתנים
  • Import שגוי: אותיות קטנות/גדולות
  • שמות שגויים: `__init__` לעומת `_init_`
  • חוסר הגדרת מחלקה: פונקציות בלי `class`
למה זה חשוב: שגיאות תחביר מונעות מהקוד לרוץ כלל!

דוגמה: שגיאות Convention

שאלה: מה לא בסדר בקוד הבא? 
from student import student
from teacher import teacher

class classroom:
    def **init**(self, class_name, homeroom_teacher):
        self.class_name = class_name
        self.homeroom_teacher = homeroom_teacher
        self.students = []

הבעיות:

  • Import שגוי: `from student import student` במקום `Student`
  • שם מחלקה שגוי: `class classroom:` במקום `Classroom`
  • __init__ שגוי: `**init**` במקום `__init__`
from student import Student
from teacher import Teacher

class Classroom:  # אות גדולה!
    def __init__(self, class_name, homeroom_teacher):  # __init__ נכון
        self.class_name = class_name
        self.homeroom_teacher = homeroom_teacher
        self.students = []

מה ללמוד:

  • PEP 8 Convention: שמות מחלקות תמיד באות גדולה (CamelCase)
  • קריאות קוד: עוזר לזהות מיד מה זה מחלקה ומה זה משתנה
  • כלי פיתוח: IDEs מזהים שגיאות לפי Convention

קטגוריה 2: שגיאות Flow Control

מה זה Flow Control?

השליטה על זרימת התוכנית - מתי פונקציות מתחילות ומסתיימות

בעיות נפוצות:

  • Return בתוך לולאה: הפונקציה מסתיימת מוקדם מדי
  • חוסר Return: פונקציה שאמורה להחזיר ערך אבל לא מחזירה
  • לוגיקה שגויה: תנאים לא נכונים
השפעה: הקוד "עובד" אבל לא עושה מה שאמור לעשות!

דוגמה: Return בתוך לולאה

שאלה: מה התוצאה של הפונקציה הבאה עם 3 תלמידים עם ציונים 80, 90, 70? 
def get_class_average(self):
    total = 0
    for student in self.students:
        total += student.get_average()
        return total / len(self.students)  # הבעיה כאן!

התוצאה: 26.67 במקום 80!

למה? הפונקציה חוזרת אחרי התלמיד הראשון בלבד!

def get_class_average(self):
    if not self.students:  # בדיקה חשובה
        return 0
    
    total = 0
    for student in self.students:
        total += student.get_average()
    return total / len(self.students)  # אחרי הלולאה!

איך להמנע:

  • חשבו על הזרימה: מתי הפונקציה אמורה להסתיים?
  • בדקו עם נתוני דוגמה: תמיד בדקו עם מספר פריטים
  • השתמשו בדיבוג: print statements לבדיקה

קטגוריה 3: מניפולציה של רשימות

הבעיה:

שינוי רשימה בזמן שעוברים עליה - אחת הבעיות הקריטיות ביותר!

מה קורה:

  • דילוג על פריטים: כשמוחקים פריט, האינדקסים משתנים

שינוי הרשימה תוך כדי מעבר עליה גורם לדילוג על פריטים או שגיאות אינדקס!

# פתרון 1: עותק של הרשימה
def remove_student(self, student_id):
    for student in self.students[:]:  # עותק!
        if student.id == student_id:
            self.students.remove(student)
            return True
    return False

# פתרון 2: רשימה חדשה
def remove_failing_students(self):
    self.students = [s for s in self.students if s.get_average() >= 60]

# פתרון 3: מעבר אחורה
def remove_failing_students(self):
    for i in range(len(self.students) - 1, -1, -1):
        if self.students[i].get_average() < 60:
            del self.students[i]

כלל זהב:

אף פעם אל תשנו רשימה שאתם עוברים עליה!

  • השתמשו בעותק: for item in my_list[:]
  • או צרו רשימה חדשה עם list comprehension
  • או עברו מאחורה לקדמה

def add_classroom(self, classroom):
    if classroom in self.classrooms:
        return False
    
    self.classrooms.append(classroom)
    
    # סינכרון חיוני!
    for student in classroom.students:
        if student not in self.all_students:
            self.all_students.append(student)
    
    for teacher in classroom.teachers:
        if teacher not in self.all_teachers:
            self.all_teachers.append(teacher)
    
    return True

תמיד סנכרנו נתונים קשורים!

  • כשמעבירים תלמיד - מעדכנים את כל הרשימות הרלוונטיות
  • חשבו על consistency בין מבני נתונים שונים

קטגוריה 6: שגיאות בהעברת פרמטרים

def transfer_student(self, student_id, new_class_name):
    student = self.find_student(student_id)
    if not student:
        return False, "Student not found"
    
    old_class = self.find_current_class(student)
    new_class = self.find_classroom(new_class_name)
    
    # בדקו מה כל פונקציה מצפה לקבל:
    old_class.remove_student(student_id)     # ID לחיפוש
    new_class.add_student(student)           # אובייקט להוספה!
    
    return True, "Transfer successful"

def get_school_average(self):
    if not self.classrooms:
        return 0.0
    
    total = 0
    valid_classes = 0
    for classroom in self.classrooms:  # נכון!
        if classroom.get_class_size() > 0:
            total += classroom.get_class_average()
            valid_classes += 1
    
    return total / valid_classes if valid_classes > 0 else 0.0
