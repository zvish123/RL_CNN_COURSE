1. השתמשו בשפה פשוטה וברורה

📚 דוגמה 1: חישוב ממוצע

🎯 הבעיה

כתוב תוכנית שמקבלת 3 ציונים ומחשבת את הממוצע שלהם

📝 פסאודו-קוד

1. התחל
2. קבל ציון1 מהמשתמש
3. קבל ציון2 מהמשתמש
4. קבל ציון3 מהמשתמש
5. חשב ממוצע = (ציון1 + ציון2 + ציון3) / 3
6. הצג את הממוצע
7. סיים

💻 קוד Python

# קבלת ציונים
ציון1 = float(input("ציון 1: "))
ציון2 = float(input("ציון 2: "))
ציון3 = float(input("ציון 3: "))

# חישוב ממוצע
ממוצע = (ציון1 + ציון2 + ציון3) / 3

# הצגת תוצאה
print(f"הממוצע: {ממוצע:.2f}")
שימו לב איך כל שורה בפסאודו-קוד הפכה לשורות קוד פשוטות ב-Python

📚 דוגמה 2: בדיקת זוגיות

🎯 הבעיה

כתוב תוכנית שבודקת אם מספר זוגי או אי-זוגי

📝 פסאודו-קוד

1. התחל
2. קבל מספר מהמשתמש
3. אם המספר מתחלק ב-2 ללא שארית:
      הצג "המספר זוגי"
   אחרת:
      הצג "המספר אי-זוגי"
4. סיים

💻 קוד Python

# קבלת מספר
מספר = int(input("הכנס מספר: "))

# בדיקת זוגיות
if מספר % 2 == 0:
    print("המספר זוגי")
else:
    print("המספר אי-זוגי")
🔑 נקודות מפתח
  • בפסאודו-קוד: "מתחלק ב-2 ללא שארית"
  • ב-Python: % 2 == 0
  • שימו לב להזחות שמראות את מבנה התנאי

📚 דוגמה 3: סכום מספרים

🎯 הבעיה

כתוב תוכנית שמחשבת את הסכום של המספרים מ-1 עד N

📝 פסאודו-קוד

1. התחל
2. קבל N מהמשתמש
3. הגדר סכום = 0
4. עבור כל מספר מ-1 עד N:
      הוסף את המספר לסכום
5. הצג את הסכום
6. סיים

💻 קוד Python

# קבלת N
N = int(input("הכנס מספר: "))

# חישוב סכום
סכום = 0
for מספר in range(1, N + 1):
    סכום += מספר

# הצגת תוצאה
print(f"הסכום: {סכום}")
בפסאודו-קוד כתבנו "עבור כל מספר מ-1 עד N" - זה הפך ל-for מספר in range(1, N + 1)
💡 בונוס: נוכל גם לכתוב בפסאודו-קוד: "חזור על התהליך N פעמים"

🔐 דוגמה 4: הצפנת Caesar

🎯 הבעיה

כתוב תוכנית להצפנת טקסט עם הצפנת Caesar (הזזת אותיות)

📝 פסאודו-קוד מפורט

1. התחל
2. קבל טקסט להצפנה מהמשתמש
3. קבל מספר הזזה (shift)
4. צור מחרוזת ריקה לתוצאה
5. עבור על כל תו בטקסט:
      אם התו הוא אות:
         - מצא את מיקום האות באלפבית (0-25)
         - הוסף את ההזזה למיקום
         - אם עברנו את 25, חזור להתחלה (modulo 26)
         - המר בחזרה לאות
         - הוסף לתוצאה
      אחרת (רווח, סימן):
         - הוסף את התו כמו שהוא לתוצאה
6. הצג את הטקסט המוצפן
7. סיים
פסאודו-קוד יכול להיות מפורט כמה שצריך - תלוי במורכבות הבעיה

🔐 הצפנת Caesar - קוד Python

💻 תרגום לקוד Python

def caesar_cipher(text, shift):
    result = ""
    
    # עבור על כל תו
    for char in text:
        if char.isalpha():
            # קביעת בסיס (A או a)
            base = ord('A') if char.isupper() else ord('a')
            
            # הצפנה עם הזזה
            encrypted = chr((ord(char) - base + shift) % 26 + base)
            result += encrypted
        else:
            # שמירת תווים לא-אותיות
            result += char
    
    return result

# שימוש בפונקציה
text = input("טקסט להצפנה: ")
shift = int(input("הזזה: "))
encrypted = caesar_cipher(text, shift)
print(f"טקסט מוצפן: {encrypted}")
🔍 השוואה
פסאודו-קוד Python
עבור על כל תו for char in text:
אם התו אות if char.isalpha():
חזור להתחלה % 26 (modulo)

🔄 דוגמה 5: לולאת while - משחק ניחושים

📝 פסאודו-קוד

1. בחר מספר אקראי בין 1-100
2. צור רשימה ריקה לניחושים
3. הגדר: נמצא = לא נכון
4. כל עוד לא נמצא:
   - קבל ניחוש מהמשתמש
   - הוסף לרשימת ניחושים
   - אם ניחוש < מספר:
        הצג "גבוה יותר"
   - אחרת אם ניחוש > מספר:
        הצג "נמוך יותר"
   - אחרת:
        הצג "נכון!"
        שנה נמצא ל-נכון
5. הצג את כל הניחושים

💻 קוד Python

import random

מספר = random.randint(1, 100)
ניחושים = []
נמצא = False

while not נמצא:
    ניחוש = int(input("נחש: "))
    ניחושים.append(ניחוש)
    
    if ניחוש < מספר:
        print("⬆️ גבוה יותר")
    elif ניחוש > מספר:
        print("⬇️ נמוך יותר")
    else:
        print("🎉 נכון!")
        נמצא = True

print(f"ניחושים: {ניחושים}")
while משמשת כאשר לא יודעים מראש כמה פעמים הלולאה תרוץ

❓ שאלות הבנה

שאלה 1

מה ההבדל העיקרי בין פסאודו-קוד לבין קוד Python?

תשובה: פסאודו-קוד אינו צריך להיות מדויק מבחינת סינטקס, והוא מתמקד בלוגיקה ובתהליך. Python דורש סינטקס מדויק וניתן להריץ אותו על המחשב.

שאלה 2

מתי כדאי להשתמש ב-while ומתי ב-for?

תשובה: for כאשר יודעים מראש כמה פעמים הלולאה תרוץ (למשל: עבור על כל פריט ברשימה). while כאשר מספר האיטרציות תלוי בתנאי דינמי (למשל: עד שהמשתמש ינחש נכון).

שאלה 3

למה חשוב לכתוב פסאודו-קוד מפורט בבעיות מורכבות?

תשובה: ככל שהבעיה מורכבת יותר, כך יותר קל לטעות או לשכוח חלקים. פסאודו-קוד מפורט עוזר לזהות את כל המקרים, לתכנן את המבנה, ולוודא שלא שכחנו כלום לפני שהתחלנו לכתוב קוד אמיתי.

שאלה 4

איך פסאודו-קוד עוזר בעבודת צוות?

תשובה: פסאודו-קוד מאפשר לכל חברי הצוות להבין את הלוגיקה ללא תלות בשפת התכנות. ניתן לדון על הגישה, לבצע code review ברמה הלוגית, ולחלק משימות בצורה ברורה יותר.
