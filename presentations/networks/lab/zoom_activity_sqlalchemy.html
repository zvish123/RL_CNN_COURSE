🔨 מה נבנה

החלפת users.json בבסיס נתונים SQLite דרך SQLAlchemy.

מודל User עם שדות:

  • id — מפתח ראשי, אוטומטי
  • username — שם משתמש ייחודי
  • password — סיסמה (בשלב הזה — טקסט)

פעולות register() ו־login() מול מסד הנתונים.

🧠 מה נלמד

  • למה מערכות אמיתיות לא שומרות נתונים רק בקובץ JSON
  • איך עובדים עם בסיס נתונים דרך Python
  • איך משתמשים ב־ORM בפרויקט אמיתי
  • איך מבצעים שאילתות בצורה מסודרת ובטוחה

🏋️ מה נתרגל

  • SQLAlchemy — הגדרת מודל וטבלה
  • session.add() — הוספת רשומה
  • filter_by() — חיפוש רשומה
  • commit() — שמירת שינויים
  • rollback() — ביטול בשגיאה
  • מודלים של טבלאות עם Column

💡 הרעיון המרכזי

כאשר מערכת גדלה, צריך שכבת נתונים מקצועית יותר, אמינה יותר ונוחה יותר לחיפוש ולעבודה.
