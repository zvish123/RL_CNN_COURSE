get_frequency_order — הכל ביחד

4 שלבים שהופכים הודעה לסדר תדירות

Python הפונקציה המלאה
def get_frequency_order(message): # שלב 1: letter → count letter_to_freq = get_letter_count(message) # שלב 2: count → [letters] freq_to_letter = {} for letter in 'ABCDEFGHIJKLMNOPQRSTUVWXYZ': freq = letter_to_freq[letter] if freq not in freq_to_letter: freq_to_letter[freq] = [letter] else: freq_to_letter[freq].append(letter) # שלב 3: המרה + מיון יורד freq_pairs = list(freq_to_letter.items()) freq_pairs.sort(key=lambda x: x[0], reverse=True) # שלב 4: חילוץ + join freq_order = [] for pair in freq_pairs: freq_order.append(''.join(pair[1])) return ''.join(freq_order)
🔢 4 שלבים
1

get_letter_count

{'A':106, 'E':175, ...}

2

היפוך → count: [letters]

{175:['E'], 20:['F','M'], ...}

3

items() + sort() יורד

[(175,'E'),(126,'T')...]

4

join → מחרוזת

"ETIASRONHLCPBDUYWGFMVKXQJZ"
