מילון מונים (Counter Dictionary)

💡 מה זה מילון מונים?

מילון מונים הוא מילון שבו המפתח הוא פריט שאנחנו סופרים (למשל אות), והערך הוא כמה פעמים הוא הופיע.

Python שלב 1: יצירת מילון מונים לאותיות
def get_letter_count(message): # מילון שסופר כמה פעמים כל אות מופיעה letter_count = {} # שלב א׳: אתחול - כל אות מתחילה מ-0 for letter in 'ABCDEFGHIJKLMNOPQRSTUVWXYZ': letter_count[letter] = 0 # שלב ב׳: ספירה - עוברים על כל אות בהודעה for letter in message.upper(): if letter in 'ABCDEFGHIJKLMNOPQRSTUVWXYZ': letter_count[letter] += 1 return letter_count
📥 קלט (Input)
"Hello World"
📤 פלט (Output) - חלקי
{'H': 1, 'E': 1, 'L': 3, 'O': 2, 'W': 1, 'R': 1, 'D': 1, ...}
⚠️ שים לב: למה אנחנו מאתחלים את כל האותיות ל-0 קודם? כדי שגם אותיות שלא מופיעות בהודעה יהיו במילון עם ערך 0, ולא נקבל KeyError.
❓ מה יהיה הערך של letter_count['Z'] אחרי הרצת הפונקציה על המחרוזת "HELLO"?
✅ נכון! כי אתחלנו את כל האותיות ל-0 בלולאה הראשונה. Z לא מופיעה ב-"HELLO" אז הערך שלה נשאר 0.

המרת מילון לרשימת זוגות

🔄 dict.items() ו-list()

בפייתון, כל מילון יכול להפוך לרשימת זוגות (tuples) בעזרת .items(). כל זוג מכיל (מפתח, ערך). אחר כך עוטפים ב-list() כדי לקבל רשימה שאפשר למיין.

# מילון מקורי
'A'106
'B'45
'E'175
'T'126
...
→
.items()
+ list()
# רשימת זוגות
[
('A', 106),
('B', 45),
('E', 175),
('T', 126),
]
Python המרת מילון לרשימת זוגות
# המילון שלנו letter_freq = {'A': 106, 'B': 45, 'E': 175} # .items() מחזיר view object של כל הזוגות items_view = letter_freq.items() # → dict_items([('A', 106), ('B', 45), ('E', 175)]) # list() הופך אותו לרשימה שאפשר למיין pairs_list = list(letter_freq.items()) # → [('A', 106), ('B', 45), ('E', 175)] # גישה לאיברים בזוג first_pair = pairs_list[0] # → ('A', 106) first_pair[0] # → 'A' (המפתח) first_pair[1] # → 106 (הערך)
💡 Tuple הוא כמו רשימה אבל בלתי ניתן לשינוי. נכתב בסוגריים עגולים: (key, value). גישה לאיברים שלו גם בעזרת אינדקס: [0] למפתח, [1] לערך.
❓ נתון: d = {'x': 5, 'y': 3}
מה יחזיר list(d.items())[1][0]?
✅ נכון! list(d.items()) → [('x',5), ('y',3)]. אינדקס [1] → הזוג השני ('y',3). אינדקס [0] מהזוג → המפתח 'y'.

מיון רשימת זוגות לפי מפתח

🗂️ list.sort(key=...)

המתודה .sort() ממיינת רשימה במקום. הפרמטר key מקבל פונקציה שאומרת לפייתון לפי מה למיין. reverse=True ממיין מהגדול לקטן.

Python מיון רשימת זוגות לפי הערך (המספר)
# פונקציית עזר שמחזירה את האיבר הראשון בזוג def get_item_at_index_zero(x): return x[0] # ---- ---- ---- ---- ---- freq_pairs = [ (106, 'A'), (175, 'E'), (45, 'B'), (126, 'T') ] # מיון יורד לפי המספר (אינדקס 0 בכל זוג) freq_pairs.sort(key=get_item_at_index_zero, reverse=True) # תוצאה: # [(175, 'E'), (126, 'T'), (106, 'A'), (45, 'B')] # אפשר גם עם lambda (פונקציה אנונימית קצרה): freq_pairs.sort(key=lambda x: x[0], reverse=True)
🔑 הכי חשוב להבין: הפרמטר key מקבל פונקציה (לא קריאה לפונקציה!). כלומר כותבים key=get_item_at_index_zero ולא key=get_item_at_index_zero(). פייתון יקרא לפונקציה הזו על כל איבר ברשימה כדי להחליט סדר.
❌ שגוי
# קוראים לפונקציה עם () freq_pairs.sort( key=get_item_at_index_zero() )
✅ נכון
# מעבירים את הפונקציה freq_pairs.sort( key=get_item_at_index_zero )
❓ נתון:
pairs = [(3, 'c'), (1, 'a'), (2, 'b')]
pairs.sort(key=lambda x: x[0])
print(pairs[0])
מה יודפס?
✅ נכון! מיון עולה (ברירת מחדל, בלי reverse=True). הקטן ביותר הוא (1, 'a'). אינדקס [0] נותן את הזוג הראשון לאחר המיון.

הכל ביחד: get_frequency_order

1

ספירת אותיות → מילון letter_to_freq

קוראים ל-get_letter_count() ומקבלים מילון מסוג {'A': 106, 'E': 175, ...}

2

היפוך המילון → freq_to_letter

מייצרים מילון הפוך: {175: ['E'], 126: ['T'], ...}. אם שתי אותיות שוות בתדירות, הן ב-list אחד.

3

המרה לרשימת זוגות + מיון

list(freq_to_letter.items()) ואז .sort(key=..., reverse=True) למיון יורד.

4

חילוץ האותיות → מחרוזת אחת

עוברים על הרשימה הממוינת ומחברים את האותיות: ''.join(freq_order)

Python get_frequency_order — הפונקציה המלאה
def get_frequency_order(message): # שלב 1: מילון letter → count letter_to_freq = get_letter_count(message) # שלב 2: מילון הפוך count → [letters] freq_to_letter = {} for letter in 'ABCDEFGHIJKLMNOPQRSTUVWXYZ': freq = letter_to_freq[letter] if freq not in freq_to_letter: freq_to_letter[freq] = [letter] else: freq_to_letter[freq].append(letter) # שלב 3: המרה + מיון יורד freq_pairs = list(freq_to_letter.items()) freq_pairs.sort(key=lambda x: x[0], reverse=True) # שלב 4: חילוץ האותיות לרשימה ואיחוד freq_order = [] for pair in freq_pairs: freq_order.append(''.join(pair[1])) return ''.join(freq_order) # דוגמה לתוצאה: # "ETIASRONHLCPBDUYWGFMVKXQJZ"

סיכום ומה למדנו

📖

מילון מונים

אתחול כל המפתחות ל-0, ואז הגדלת ערך ב-1 לכל מופע. מניע פגיעות KeyError.

🔄

.items()

מחזיר view של זוגות (key, value). עוטפים ב-list() לקבלת רשימה שאפשר למיין.

🗂️

מיון עם key

.sort(key=func) — הפונקציה אומרת לפי מה למיין. reverse=True = מהגדול לקטן.

🔗 הקוד המחבר הכל יחד
# 1. מילון מונים counter = {} for item in collection: counter[item] = counter.get(item, 0) + 1 # 2. המרה לרשימת זוגות pairs = list(counter.items()) # 3. מיון יורד לפי הערך pairs.sort(key=lambda x: x[1], reverse=True) # 4. שימוש בתוצאה print(f"הכי שכיח: {pairs[0][0]} ({pairs[0][1]} פעמים)")
