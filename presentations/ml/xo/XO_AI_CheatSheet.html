🐍 XO-AI Developer Cheat Sheet

ריכוז פקודות, פרמטרים ולוגיקה לפרויקט הבינה המלאכותית

⚙️ הגדרות ניסוי (ExperimentConfig)

ניהול הפרמטרים של ה-AI באמצעות @dataclass.

# יצירת קונפיג חדש config = ExperimentConfig( name="Fast_Learner", num_games=10000, gamma=0.95, exploration_rate=0.2 )
Gamma
חשיבות העתיד (0 עד 1). ערך גבוה = ראייה לטווח רחוק.
Exploration
סיכוי למהלך אקראי (Epsilon). עוזר לגלות אסטרטגיות.
Rewards
ניקוד על פעולות (Win, Loss, Tie). מעצב את "אופי" ה-AI.

🧠 לוגיקה ואסטרטגיה (Heuristics)

סדר העדיפויות בקבלת החלטות ב-enhanced_game.py:

  1. ניצחון מיידי: check_winning_move()
  2. חסימת יריב: check_blocking_move()
  3. תפיסת המרכז: משבצת (1, 1)
  4. פינות: (0,0), (0,2), (2,0), (2,2)
# דוגמה לבדיקת חסימה if self.board[1][1] == " ": return self.make_move(1, 1)

🚀 אוטומציה (ExperimentRunner)

הרצת סדרת ניסויים מתוך קובץ CSV.

runner = ExperimentRunner() # טעינת רשימת ניסויים configs = runner.load_experiments_from_csv() # הרצה וקבלת סטטיסטיקות results = runner.run_multiple_experiments(configs)

שים לב: ה-Runner שומר את התוצאות אוטומטית לתיקיית experiment_results.

💎 פייתון: טיפים וכלים

f-strings
f"Result: {score}"
Dicts
{state: score} (מילון הלמידה)
StaticMethod
פונקציה ששייכת למחלקה ולא לאובייקט.
Reversed
reversed(history) - לעדכון מהסוף להתחלה.

🔍 טיפים לדיבאגינג (למה ה-AI לא לומד?)

  • האם Gamma נמוך מדי? (מתחת ל-0.5 ה-AI "רואה" רק את הסוף).
  • האם ה-num_games קטן מדי? (פחות מ-1000 משחקים לא מספיקים ללמידה).
  • האם הקובץ JSON נשמר ונטען מחדש? (בדוק בתיקיית הטורניר).
  • האם ה-Exploration Rate הוא 0? (ה-AI לא ינסה דברים חדשים).
