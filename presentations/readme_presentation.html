## התקנה ### דרישות מערכת: - Node.js 14.0 או גרסה חדשה יותר - npm או yarn ### צעדי התקנה: 1. שכפל את המאגר: ```bash git clone https://github.com/username/project.git ``` 2. עבור לתיקיית הפרויקט: ```bash cd project ``` 3. התקן תלויות: ```bash npm install ``` 4. הפעל את הפרויקט: ```bash npm start ```
חוקי הזהב

תמיד בדוק את הוראות ההתקנה על מחשב חדש! מה שעובד אצלך לא בהכרח יעבוד אצל אחרים.

דוגמאות שימוש שמזמינות לפעולה 🚀

הראה, אל תספר!

## שימוש ### דוגמה בסיסית: ```javascript const calculator = new Calculator(); const result = calculator.add(5, 3); console.log(result); // 8 ``` ### דוגמה מתקדמת: ```javascript calculator .add(10) .multiply(2) .subtract(5); console.log(calculator.getResult()); // 15 ``` ### שימוש מהשורת פקודה: ```bash calc "2 + 2 * 3" # Result: 8 ```
