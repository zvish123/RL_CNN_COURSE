🎯 מה נלמד?

במטלה זו תבנו מערכת מבוזרת בת שני צדדים: שרת המנהל את החיפוש ומחלק טווחי מחרוזות ללקוחות, ולקוחות המריצים חישוב MD5 מקבילי בכמה Threads בו-זמנית. המטרה: מציאת מחרוזת plaintext שמייצרת Hash נתון — בדיוק כפי שעושים כלי Brute-Force אמיתיים בעולם אבטחת המידע.

🧵 Threads

שימוש ב-threading לחלוקת טווחי חיפוש בין תהליכונים מרובים בכל לקוח.

🌐 Sockets

תקשורת TCP בין שרת ולקוחות: קבלת משימה, ביצוע, דיווח תוצאה.

🔐 MD5 Hashing

שימוש ב-hashlib לחישוב ה-Hash וניתוח המושג "חד-כיווניות".

⚙️ ניהול משאבים

Lock, Event, ניצול CPU יעיל, מניעת race conditions.

import socket, threading, hashlib, itertools

class MD5Client:
    def __init__(self, host='127.0.0.1', port=9999, num_threads=4):
        self.host = host
        self.port = port
        self.num_threads = num_threads
        self.stop_event = threading.Event()   # מאפשר עצירת כל ה-Threads
        self.result = None
        self.lock = threading.Lock()

    def compute_md5(self, text: str) -> str:
        return hashlib.md5(text.encode()).hexdigest()

    def search_range(self, candidates, target_hash):
        """בדוק כל מחרוזת ברשימה מול ה-hash. עצור אם נמצא."""
        for candidate in candidates:
            if self.stop_event.is_set():
                return
            if self.compute_md5(candidate) == target_hash:
                with self.lock:
                    self.result = candidate
                self.stop_event.set()   # אות לכל ה-Threads
                return

🧵 דוגמה לחלוקת עבודה בין Threads

python 
import threading, itertools, string

def split_work(all_candidates, num_threads):
    """חלק איטרטור לרשימות — chunk לכל thread."""
    all_list = list(all_candidates)
    chunk_size = len(all_list) // num_threads
    return [all_list[i*chunk_size:(i+1)*chunk_size]
            for i in range(num_threads)]

# כל המחרוזות באורך 1-3 תווים מהאלפבית
# מרחב חיפוש: כל המספרים בני 9 ספרות
start, end = 100_000_000, 999_999_999
all_numbers = range(start, end + 1)

# חלוקה שווה בין 4 threads
chunks = split_work(all_numbers, 4)
# כל chunk: ~225 מיליון מספרים
