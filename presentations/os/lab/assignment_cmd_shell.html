חלק א — בניית Shell מינימלי בפייתון

// מטרה

בנה תוכנית Python שמדמה Shell/CMD — לולאה שמקבלת קלט מהמשתמש, מפרסרת את הפקודה, ומבצעת אותה. ה-Shell יפעיל גם פקודות built-in (שכתבת) וגם פקודות מערכת חיצוניות.

REQ-01  ·  לולאת REPL + Prompt
  • הצג Prompt בפורמט: user@host:path> (צבעוני עם ANSI codes)
  • קבל שורת קלט, פרסר לפקודה + ארגומנטים
  • הדפס הודעת שגיאה ברורה אם הפקודה לא קיימת
  • הפקודה exit / quit יוצאת מה-Shell בנימוס
# מבנה בסיסי של ה-Shell
def main_loop():
    while True:
        prompt = build_prompt()        # user@host:cwd
        line   = input(prompt).strip()
        if not line: continue
        tokens = parse_line(line)        # פיצול חכם
        dispatch(tokens)               # built-in / external

REQ-03  ·  פקודות חיצוניות + subprocess
  • פקודות שאינן built-in → הפעל עם subprocess.run()
  • הצג stdout ו-stderr בנפרד
  • שמור את קוד החזרה ($?) ואפשר לבדוק אותו
  • Timeout: אם תהליך חיצוני תקוע מעל N שניות — הצע להרוג
import subprocess
def run_external(tokens):
    result = subprocess.run(tokens, capture_output=True, text=True)
    if result.stdout: print(result.stdout, end='')
    if result.stderr: print(f"\033[91m{result.stderr}\033[0m", end='')
    env['?'] = str(result.returncode)   # $? כמו bash
REQ-04  ·  Piping בין פקודות
  • זיהוי תו | בשורת הפקודה
  • חיבור stdout של פקודה שמאלית ל-stdin של ימנית
  • תמיכה בשרשרת ארוכה: cmd1 | cmd2 | cmd3
  • Pipeline עובד גם בין built-in לחיצונית וגם להיפך
# דוגמה לפקודות שצריכות לעבוד:
myshell> ls | cat -n
myshell> cat log.txt | grep error
myshell> echo hello world | python -c "import sys; print(sys.stdin.read().upper())"
REQ-05  ·  משתני סביבה
  • שמור Dict פנימי של משתני סביבה (ירש מ-os.environ בהתחלה)
  • הרחבת משתנים בפקודות: echo $HOME, cd $MYDIR
  • פקודת set X=value מגדירה משתנה; set בלבד מציג הכל
  • משתנים מיוחדים: $? (קוד חזרה), $0 (שם Shell), $PWD
  • משתנים מועברים לפקודות חיצוניות (env= ב-subprocess)
REQ-06  ·  היסטוריה + השלמה
  • שמור היסטוריה של עד 500 פקודות בזיכרון ובקובץ ~/.myshell_history
  • ניווט בחצים Up/Down לעיון בהיסטוריה (readline module)
  • חיפוש היסטוריה: history | grep cd
  • חזרה על פקודה: !3 (פקודה מספר 3), !! (אחרונה)
★ BONUS-01  ·  מנגנון פלאגינים
  • תמיכה בטעינת פקודות מקבצי Python חיצוניים בתיקיית plugins/
  • כל פלאגין מגדיר: COMMAND_NAME, run(args, env), HELP
  • הדגמה: כתוב פלאגין sysinfo.py שמציג מידע על המערכת
# plugins/sysinfo.py
COMMAND_NAME = "sysinfo"
HELP = "הצג מידע על המערכת"

def run(args, env):
    import platform
    print(f"OS: {platform.system()} {platform.release()}")
    print(f"Python: {platform.python_version()}")
★ BONUS-02  ·  Redirection + Script Mode
  • תמיכה ב-Redirection: echo hi > out.txt, cat < in.txt, ls >> log.txt
  • Script Mode: myshell.py script.sh מריץ קובץ שורה אחרי שורה
  • תמיכה בהערות: שורות שמתחילות ב-# מתעלמים מהן

חלק ב — מחקר Sysinternals על ה-Shell שלך

// מטרה

לאחר שה-Shell שלך עובד — השתמש ב-Sysinternals Suite לחקור אותו בזמן ריצה. ענה על ארבעת שאלות המחקר הבאות ותעד את ממצאיך עם screenshots + ניתוח.

🔬 מחקר 1 — Process Explorer: מה קורה בזמן ריצה?

הרץ את ה-Shell שלך ובצע את הצעדים הבאים ב-Process Explorer:

  1. אתר את תהליך python.exe של ה-Shell שלך בעץ התהליכים
  2. פתח את ה-Properties ובחן את לשונית Threads — כמה threads? מה כל אחד עושה?
  3. לשונית Handles — מה ה-Shell שלך מחזיק? האם יש File handles? Registry?
  4. לשונית DLLs — רשום את 5 ה-DLLs המעניינות שנטענות. מה תפקיד כל אחת?
  5. הרץ פקודה חיצונית (למשל dir) — מה קורה לעץ התהליכים? מתי ה-Child נעלם?

🔬 מחקר 2 — Process Monitor: מה ה-Shell נוגע?

הפעל ProcMon (Ctrl+L לסינון) וצפה בפעולות ה-Shell שלך:

  1. סנן לפי Process Name = python.exe
  2. הרץ את הפקודה cd Desktop — אילו Registry operations רואים?
  3. הרץ cat myfile.txt — מה הסדר של File operations? (CreateFile, ReadFile, CloseFile)
  4. הרץ set MYVAR=hello — האם ה-Shell כותב לרגיסטרי? למה כן/לא?
  5. מה הנתיבים שה-Shell מחפש בהם קבצי Python modules? (חפש PATH NOT FOUND)

🔬 מחקר 3 — TCPView: האם ה-Shell מתחבר לרשת?

פתח TCPView ובחן:

  1. בזמן ריצה רגילה של ה-Shell — האם פייתון מחזיק ports פתוחים? מדוע כן/לא?
  2. הרץ pip install requests מתוך ה-Shell שלך — מה TCPView מראה? לאן מתחבר?
  3. אם כתבת פקודת wget / curl — נטר את החיבור
  4. בדוק: האם Python בטעינה פותח Socket כלשהו גם ללא בקשת רשת?

// מה יעזור לך
  • השתמש ב-shlex.split() לפרסינג — מטפל בציטוטים נכון
  • readline module — השלמה אוטומטית ב-5 שורות
  • בנה כל פקודה כ-class או function נפרדת — קל לבדוק
  • כתוב tests קטנים לכל פקודה לפני שמחברים הכל
  • ProcMon — Filter > Include > Process Name IS python.exe
// שגיאות נפוצות
  • אל תשנה os.environ ישירות — שמור Dict משלך
  • Pipe עם subprocess — השתמש ב-PIPE flag, לא ב-print
  • פקודות עם רווחים בשמות קבצים — shlex פותר זאת
  • Ctrl+C בתהליך חיצוני — טפל ב-KeyboardInterrupt
  • ProcMon מאט את המערכת — הפעל רק בזמן הניסוי הספציפי
// Skeleton — התחל מכאן
import os, sys, shlex, subprocess, readline

HISTORY  = []
ENV      = dict(os.environ)               # עותק של סביבה
ENV['?']  = '0'
BUILTINS = {}                             # dict: name -> function

def register(name):                       # decorator לרישום built-in
    def wrapper(fn):
        BUILTINS[name] = fn; return fn
    return wrapper

@register('cd')
def cmd_cd(args):
    path = args[0] if args else os.path.expanduser('~')
    try:
        os.chdir(path); ENV['PWD'] = os.getcwd()
    except FileNotFoundError:
        print(f"cd: {path}: לא נמצא")

def expand_vars(token):                   # הרחבת $VAR
    for k, v in ENV.items():
        token = token.replace(f'${k}', v)
    return token

def dispatch(tokens):
    if not tokens: return
    tokens = [expand_vars(t) for t in tokens]
    cmd, *args = tokens
    if cmd in BUILTINS:
        BUILTINS[cmd](args)
    else:
        result = subprocess.run(tokens, env=ENV)
        ENV['?'] = str(result.returncode)

def main():
    while True:
        try:
            line = input(f"\033[96m{os.getcwd()}\033[0m> ").strip()
            if not line: continue
            HISTORY.append(line)
            # TODO: handle pipes here
            dispatch(shlex.split(line))
        except KeyboardInterrupt: print()
        except EOFError: break

if __name__ == '__main__': main()
// לוח זמנים מומלץ
