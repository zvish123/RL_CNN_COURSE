🎯 מסננים (Filters/Kernels)

🔍

מטריצות קטנות שמזהות תבניות

• גודל טיפוסי: 3×3, 5×5

• ערכים הנלמדים באימון

⚡ פונקציית הפעלה (ReLU)

📈

מוחקת ערכים שליליים

• ReLU(x) = max(0, x)

• מוסיפה אי-לינאריות

• מאפשרת למידה מורכבת

⚡ פונקציית הפעלה - ReLU

📊 הגרף של ReLU

הנוסחה: ReLU(x) = max(0, x)

• אם x > 0 ← תוצאה: x

• אם x ≤ 0 ← תוצאה: 0

📋 סיכום - מבנה שכבת קונבולוציה

🎯 מה למדנו?

• מסננים מזהים תבניות

• ReLU מוחק ערכים שליליים

• Pooling מפחית ממדים

• בניית שכבה מלאה

🔑 נקודות מפתח

• כל רכיב משרת מטרה ספציפית

• הפחתת ממדים חוסכת משאבים

• המבנה חוזר בכל שכבה

• למידה היררכית של תכונות

🚀 הבא בתור

• בניית רשת CNN מלאה

• שכבות fully connected

• אימון ובדיקה

• יישומים מעשיים
