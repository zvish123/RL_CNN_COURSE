פרסינג ידני עם split()

מנתחים את המחרוזת לבד, שלב אחרי שלב
    

      

      Python
    

    

text = "[(1,2),(3,8), (7,1), (9,4)]"

# שלב 1: הסר סוגריים חיצוניים [ ]
cleaned = text.strip()[1:-1]
# "(1,2),(3,8), (7,1), (9,4)"

# שלב 2: פצל לפי "),("
parts = cleaned.split("),(")
# ["(1,2", "3,8", " 7,1", " 9,4)"]

# שלב 3: נקה כל חלק ובנה טאפל
result = []
for part in parts:
    part = part.strip("() ")
    a, b = part.split(",")
    result.append((int(a.strip()), int(b.strip())))

print(result)  # [(1, 2), (3, 8), (7, 1), (9, 4)]
1
מה עושה strip()[1:-1]?
▶
"[(1,2),(3,8)]" → [1:-1] → "(1,2),(3,8)"
[1:-1] הוא slicing — חותך את התו הראשון ([) ואת האחרון (]). strip() מנקה רווחים לפני.
2
מה עושה split("),(")?
▶
"(1,2),(3,8),(7,1)" → split by ),( → ["(1,2", "3,8", "7,1)"]
מפצל בנקודות שיש ),( — זה מפריד בין הטאפלים. שימו לב שחלק מהסוגריים נשארים!
3
מה עושה strip("() ")?
▶
"(1,2" → strip("() ") → "1,2"
מנקה את הסוגריים הנותרים ורווחים מכל צד של המחרוזת. הארגומנט "() " אומר: נקה את הסימנים האלה.

Regex — ביטויים רגולריים

מתארים תבנית ומחפשים אותה

import re

text = "[(1,2),(3,8), (7,1), (9,4)]"

# חיפוש התבנית: (מספר, מספר)
pairs = re.findall(r'\((\d+),\s*(\d+)\)', text)
# [('1','2'), ('3','8'), ('7','1'), ('9','4')]

# המרה למספרים שלמים
result = [(int(a), int(b)) for a, b in pairs]

print(result)  # [(1, 2), (3, 8), (7, 1), (9, 4)]

eval — הדרך המסוכנת

עובד — אבל לעולם אל תשתמשו בזה עם קלט חיצוני!
    

      

      Python
    

    

text = "[(1,2),(3,8), (7,1), (9,4)]"

# עובד! שורה אחת בלי imports
result = eval(text)

print(result)  # [(1, 2), (3, 8), (7, 1), (9, 4)]
print(type(result))  # <class 'list'>
⚠️ למה eval מסוכן?
eval() לא מבדילה בין נתון לבין קוד — היא פשוט מריצה כל מה שתתנו לה. אם המשתמש שלכם שולח קלט זדוני, זה ירוץ על השרת שלכם!

ast.literal_eval — המנצחת

בטוח, קצר, אמין — הדרך הנכונה
    

      

      Python
    

    

import ast

text = "[(1,2),(3,8), (7,1), (9,4)]"

result = ast.literal_eval(text)

print(result)        # [(1, 2), (3, 8), (7, 1), (9, 4)]
print(type(result))  # <class 'list'>
print(result[0])     # (1, 2)
print(type(result[0])) # <class 'tuple'>
