מה זה ה-Registry?

ה-Registry הוא מאגר נתונים מרכזי של Windows המכיל את כל הגדרות המערכת והתוכנות

📁 HKEY_CURRENT_USER (HKCU) - הגדרות המשתמש הנוכחי
📁 HKEY_LOCAL_MACHINE (HKLM) - הגדרות כלל המערכת
📁 HKEY_CLASSES_ROOT (HKCR) - קשרי קבצים ותוכנות
📁 HKEY_USERS (HKU) - כל פרופילי המשתמשים
📁 HKEY_CURRENT_CONFIG (HKCC) - הגדרות חומרה

⚠️ זהירות!

תמיד בצעו Export לפני עריכת ה-Registry

תרגיל Autoruns

🔍 משימה 2: חקירת תוכנות אוטומטיות

1. הריצו "autoruns" כמנהל מערכת
2. בטאב "Everything" - סקרו תוכנות אוטומטיות
3. מצאו את ה-"Scheduled Tasks"
4. Options → Scan Options → ✓ VirusTotal
5. Options → Scan Options → ✓ Verify Signatures

🚨 אתרעה אבטחה

תוכנה לא חתומה + אזעקות VirusTotal = חשד לנוזקה!

תרגיל מודרך - Procmon

🔍 משימה 4: מציאת הגדרות Notepad ברגיסטרי

1. פתחו Procmon והתחילו הקלטה
2. פתחו Notepad ושנו גודל וסוג פונט
3. סגרו Notepad ועצרו הקלטה
4. פילטר: "Process Name is notepad.exe"
5. פילטר נוסף: רק Registry events
6. פילטר אחרון: "Operation is RegSetValue"
7. קליק ימני → "Jump To" ברגיסטרי
// טיפים לפילטרים:
• Include - הציג רק...
• Exclude - הסתר...
• Ctrl+L - עורך פילטרים
• Ctrl+F - חיפוש טקסט
• Space - השהה/המשך הקלטה

🎯 מה גילינו?

Notepad שומר הגדרות פונט בנתיב הרגיסטרי:
HKCU\Software\Microsoft\Notepad

🔍 תרחיש: חשד להדבקה בנוזקה

🛡️ משימה 5: חקירת אבטחה

1. Autoruns: חפשו תוכנות לא חתומות
2. בדקו עם VirusTotal תוכנות חשודות
3. Process Explorer: מצאו processes עם צריכת CPU גבוהה
4. בדקו חתימות דיגיטליות
5. Procmon: עקבו אחר פעילות רשת חשודה
סיכון גבוה - תוכנה לא חתומה + VirusTotal alerts
סיכון בינוני - תוכנה לא חתומה ללא alerts
בטוח - תוכנה חתומה מיצרן מוכר
