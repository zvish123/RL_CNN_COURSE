#3
בניית הבסיס
למסירה: מפתח 1 (תמר) עדיפות: גבוהה זמן: 15 דקות
משימה: יצירת מבנה HTML בסיסי לדף הציטוטים

שלבי ביצוע:

  1. צור branch חדש: foundation
  2. עבור לbranch החדש
  3. צור קובץ הוסף index.html בסיסי עם מבנה HTML5 בסיסי
  4. כלול: doctype, html, head, meta, title, body
  5. השתמש בעברית כשפה (lang="he" dir="rtl")
  6. commit + push origin foundation
  7. צור Pull Request
git branch foundation
git checkout foundation
# צור index.html
git add index.html
git commit -m "Add basic HTML structure"
git push origin foundation
⏳ אל תמזג! חכה לאישור מנהל הגרסה
