דוגמאות בלחיצת כפתור: חשיבה חישובית בקריפטוגרפיה

במפגש מורים חשוב להראות לא רק את שמות המרכיבים, אלא כיצד הם נראים בפעילות כיתה אמיתית: סיפור, דיון, קוד, מסקנות והעברה לנושא הבא.

🎲 אי־דטרמיניזם — תוקף לא צפוי

רעיון כיתתי: מערכת IDS מגינה על שרת. התוקף יכול לבחור פורט אקראי, ולכן ההגנה צריכה להתמודד עם קלט שלא ידוע מראש.

allowed_ports = [80, 443, 22] all_possible_ports = [21, 22, 25, 80, 443, 3389, 8080, 9001] attacker_port = random.choice(all_possible_ports) if attacker_port in allowed_ports: print("connection allowed") else: print("connection blocked")

מסר פדגוגי: התוקף אינו צפוי, אך כלל ההגנה חייב להיות קבוע וברור. זו הזדמנות להשוות בין Whitelist ל־Blacklist.

מסר פדגוגי: במקום לשאול “מה קרה קודם?”, שואלים “מה היה חייב לקרות כדי שהתוצאה תתרחש?”. זה מתאים גם לפענוח צפנים וגם לחקירת אירועי סייבר.

🔍 זיהוי דפוסים — לוגים חשודים

רעיון כיתתי: התלמידים מקבלים כמה לוגים ומנסחים כלל שמבדיל בין פעילות רגילה לבין ניסיון Brute Force.

מסר פדגוגי: לפני כתיבת הקוד התלמידים מגלים את החוק בעצמם. הקוד הוא ניסוח אלגוריתמי של דפוס שהם כבר זיהו.

דוגמאות נוספות בלחיצת כפתור: איך הופכים מושג לפעילות?

🎭 הפשטה

להפריד בין הסיפור לבין המודל הקריפטוגרפי.

סיפור: Alice שולחת הודעה ל־Bob ו־Eve מאזינה.

הפשטה: plaintext → encryption(key) → ciphertext → decryption(key)

שאלת מורה: אילו פרטים בסיפור אינם חשובים לאלגוריתם?

🧩 פירוק

לפרק מערכת הצפנה לשלבים קטנים.

משימה: בניית צופן ויז'נר.

פירוק: ניקוי טקסט → חישוב הזזה לפי מפתח → הצפנת אות → חיבור תוצאה → פענוח → בדיקות.

🔄 שינוי ייצוג

לעבור מטקסט למספרים, לביטים ול־Hash.

"A" → 65 → 01000001 "secret" → bytes → SHA-256 → digest

שאלת מורה: באיזה ייצוג הכי קל להסביר? באיזה ייצוג הכי קל לחשב?

🌐 הכללה

מעבר מצופן אחד למשפחה של צפנים.

לאחר צופן קיסר שואלים: מה קורה אם כל אות מקבלת הזזה אחרת? מכאן מגיעים לצופן ויז'נר ולרעיון של מפתח ארוך יותר.

למידה מטעויות: להפוך שגיאת קוד להזדמנות הוראה

למידה מטעויות אינה “תיקון טכני”. היא כלי דידקטי: התלמיד מזהה סימפטום, מסביר את מקור השגיאה, מתקן, ואז מנסח כלל למקרים הבאים.

מקרה 2 — רצף if שגורם ל־else לא נכון

# שגוי if alien_type == "green": ... if alien_type == "red": ... if alien_type == "blue": ... else: print("חייזר לא מזוהה") # נכון if alien_type == "green": ... elif alien_type == "red": ... elif alien_type == "blue": ... else: print("חייזר לא מזוהה")
