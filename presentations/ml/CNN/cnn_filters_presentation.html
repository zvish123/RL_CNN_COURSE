import cv2 import numpy as np import matplotlib.pyplot as plt # ── טעינה והכנה ────────────────────────────────────────── img = cv2.imread("missile.jpg") img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB) # לתצוגה img_gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY) # לפילטרים # ── הגדרת Kernels ───────────────────────────────────────── blur = (1/9) * np.ones((3,3)) sharpen = np.array([[ 0,-1, 0], [-1, 5,-1], [ 0,-1, 0]]) horiz = np.array([[-1,-1,-1], [ 0, 0, 0], [ 1, 1, 1]]) sobel_v = np.array([[-1, 0, 1], # ← Custom Filter שלנו! [-2, 0, 2], [-1, 0, 1]]) sobel_h = np.array([[-1,-2,-1], [ 0, 0, 0], [ 1, 2, 1]]) # ── Combined Sobel (עוצמת קצה בכל כיוון) ──────────────── gx = cv2.filter2D(img_gray, cv2.CV_64F, sobel_v) gy = cv2.filter2D(img_gray, cv2.CV_64F, sobel_h) combined = cv2.convertScaleAbs(cv2.magnitude(gx, gy)) # ── הפעלת פילטרים ──────────────────────────────────────── images = [img_rgb, cv2.filter2D(img_gray, -1, blur), cv2.filter2D(img_gray, -1, sharpen), cv2.filter2D(img_gray, -1, horiz), cv2.filter2D(img_gray, -1, sobel_v), cv2.filter2D(img_gray, -1, sobel_h), combined] titles = ["Original","Blur","Sharpen","Horiz","Sobel-V","Sobel-H","Combined"] # ── הצגה ───────────────────────────────────────────────── fig, axes = plt.subplots(2, 4, figsize=(20, 10)) for ax, img_, title in zip(axes.flatten(), images, titles): ax.imshow(img_, cmap=None if len(img_.shape)==3 else "gray") ax.set_title(title); ax.axis("off") plt.tight_layout(); plt.show()

1. למה OpenCV צריך cvtColor(BGR2RGB) לפני הצגה ב-Matplotlib?

כי Matplotlib לא תומך בתמונות צבע
כי OpenCV קורא בסדר BGR ו-Matplotlib מצפה ל-RGB
כי זה משפר את איכות הצבעים

2. Kernel שסכום משקולותיו = 0 עושה:

טשטוש
חידוד
זיהוי שינויים / קצוות – אחיד → שחור, שינוי → לבן

3. מה מייחד את פילטר Emboss?

הוא מזהה קצוות אנכיים בלבד
הוא מדגיש הבדלי צבע אלכסוניים ויוצר אפקט תלת-ממד
הוא שומר על הצבע המקורי

4. מהו Combined Sobel ולמה הוא שימושי?

ממוצע של Sobel-X ו-Y
מגניטודה √(Gx²+Gy²) – מוצא קצוות בכל כיוון שהוא
הכפלת שני ה-Kernel-ים
מה למדנו היום? 🎓
🔢

מספרים = כוונה

כל ערך ב-Kernel מקודד תכונה ספציפית: כיוון, עוצמה, שכן רלוונטי.

➕➖

חיובי / שלילי / אפס

חיובי=חזק, שלילי=החלש, אפס=התעלם. שלושה מצבים יוצרים כל אפקט.

Σ

סכום קובע הכל

Σ=1 → שמירת בהירות. Σ=0 → זיהוי שינויים. Σ>1 → הגברה.

🎨

BGR ≠ RGB

OpenCV קורא BGR. תמיד cvtColor לפני matplotlib. לפילטרים – השתמש ב-GRAY.

⚡

Sobel > Edge פשוט

×2 לאמצע → עמיד לרעש ומדויק. Combined Sobel = √(Gx²+Gy²).
