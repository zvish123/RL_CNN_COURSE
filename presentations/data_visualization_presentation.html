🐍 היכרות עם Seaborn

מה זה Seaborn?

ספריית Python מתקדמת ליצירת גרפים סטטיסטיים יפים ואינפורמטיביים

✨ יתרונות Seaborn:

  • פשטות: קוד קצר ליצירת גרפים מורכבים
  • יופי: עיצוב מקצועי ואסתטי
  • אינטגרציה: עובד מעולה עם Pandas
  • גמישות: אפשרויות התאמה רבות

📦 התקנה ויבוא:

# התקנה (רק פעם אחת)
pip install seaborn

# יבוא בתחילת הקוד
import seaborn as sns
import pandas as pd
import matplotlib.pyplot as plt

# הגדרת נושא
sns.set_theme()
💡 טיפ: Seaborn בנוי על גבי matplotlib, אז אפשר לשלב ביניהם!

📊 גרפים בסיסיים

📈 Histogram - היסטוגרמה

מתי נשתמש: להבנת התפלגות של משתנה בודד

sns.histplot(data['SalePrice'], bins=30)
plt.title('התפלגות מחירי דירות')
plt.show()
🔢 Count Plot - ספירת קטגוריות

מתי נשתמש: לספירת ערכים בשדה קטגוריאלי

sns.countplot(data['SaleCondition'])
plt.title('סוגי תנאי מכירה')
plt.show()
📉 Box Plot - תיבה ושפם

מתי נשתמש: להשוואת התפלגויות בין קבוצות

sns.boxplot(x='Neighborhood', y='SalePrice')
plt.xticks(rotation=45)
plt.show()
🔗 Scatter Plot - פיזור

מתי נשתמש: לזיהוי קשר בין שני משתנים מספריים

sns.scatterplot(x='1stFlrSF', y='SalePrice')
plt.title('קשר בין גודל למחיר')
plt.show()
💡 זכרו: כל סוג גרף עונה על שאלה אחרת - בחרו את המתאים!

🔗 ניתוח קורלציות

מה זה קורלציה?

מדידה של עוצמת הקשר הליניארי בין שני משתנים מספריים

  • 1.0: קשר חיובי מושלם
  • 0.0: אין קשר ליניארי
  • -1.0: קשר שלילי מושלם

🔍 חישוב קורלציות בקוד:

# חישוב קורלציה עם מחיר הדירה
correlations = housing.corr()['SalePrice'].sort_values(ascending=False)
print(correlations)

# הצגת התוצאות החזקות ביותר
print("הקשרים החזקים ביותר:")
print(correlations.head(10))

🏠 תוצאות מהדוגמה שלנו:

  • OverallQual (0.79): איכות כללית - קשר חזק מאוד!
  • GrLivArea (0.71): שטח מגורים - קשר חזק
  • GarageCars (0.64): מספר מכוניות בחניה - קשר בינוני חזק

📊 הצגה ויזואלית:

# יצירת גרף פיזור עם קו מגמה
sns.scatterplot(x='OverallQual', y='SalePrice', data=housing)
sns.regplot(x='OverallQual', y='SalePrice', data=housing, scatter=False)
plt.title('קשר בין איכות כללית למחיר')
plt.show()

🎨 ויזואליזציות מתקדמות

🎯 Joint Plot

משלב גרף פיזור עם היסטוגרמות של שני המשתנים

sns.jointplot(x='1stFlrSF', y='SalePrice', 
              data=housing, kind='reg')
plt.show()
🗂️ Pair Plot

מטריצה של גרפים המשווה כל משתנה לכל משתנה

sns.pairplot(housing[['SalePrice', 
            'OverallQual', '1stFlrSF']])
plt.show()
🌡️ Heatmap

מפת חום המציגה קורלציות בין כל המשתנים

plt.figure(figsize=(12, 8))
sns.heatmap(housing.corr(), 
            annot=True, cmap='coolwarm')
plt.show()
📊 Violin Plot

משלב Box Plot עם הצגת צפיפות ההתפלגות

sns.violinplot(x='Neighborhood', 
               y='SalePrice', data=housing)
plt.xticks(rotation=45)
plt.show()

💡 איך לבחור את הגרף הנכון?

  • משתנה בודד: Histogram, Count Plot
  • שני משתנים מספריים: Scatter Plot, Joint Plot
  • מספרי מול קטגוריאלי: Box Plot, Violin Plot
  • מספר משתנים: Pair Plot, Heatmap

🚀 קוד מקצועי:

# הגדרת סגנון יפה
plt.figure(figsize=(12, 8))
sns.set_style("whitegrid")
sns.set_palette("husl")

# יצירת גרף עם כותרות
ax = sns.scatterplot(x='1stFlrSF', y='SalePrice', data=housing)
ax.set_title('הקשר בין שטח הבית למחיר המכירה', fontsize=16, fontweight='bold')
ax.set_xlabel('שטח קומה ראשונה (רגל מרובע)', fontsize=12)
ax.set_ylabel('מחיר מכירה ($)', fontsize=12)

# הוספת קו מגמה
sns.regplot(x='1stFlrSF', y='SalePrice', data=housing, scatter=False, ax=ax)

plt.tight_layout()
plt.show()
🎯 זכרו: הגרף צריך לספר סיפור - לא רק להציג נתונים!
