חתימה דיגיטלית: JSON Web Token (JWT)

מה זה JWT?

JWT הוא תקן מבוסס JSON להעברת מידע בצורה מאובטחת בין צדדים (כגון שרת ולקוח). המידע ב-JWT חתום דיגיטלית, כך שניתן לאמת את תקינותו ואת מקורו.

סיכום זרימה כללית

  1. הלקוח שולח פרטי התחברות (username + password)
  2. השרת מאמת ויוצר JWT (נחתם עם SECRET_KEY שנמצא רק בשרת)
  3. השרת שולח את ה-JWT ללקוח
  4. הלקוח שומר את ה-JWT ושולח אותו בכל בקשה עתידית
  5. השרת מאמת את החתימה בכל בקשה — ללא שאילתת DB נוספת

מבנה JWT

JWT מורכב משלושה חלקים, המופרדים בנקודות:

Header (כותרת)

מידע על סוג האלגוריתם בו נעשה שימוש לחתימה:

{
    "alg": "HS256",
    "typ": "JWT"
}

Payload (מטען)

מכיל נתונים המועברים עם האסימון:

{
    "user_id": "12345",
    "role": "admin",
    "exp": 1712345678
}

Signature (חתימה)

נועדה להבטיח שהאסימון לא שונה לאחר יצירתו. נוצר על ידי שילוב ה-Header, ה-Payload, ומפתח סודי:

HMACSHA256(base64UrlEncode(header) + "." + base64UrlEncode(payload), secret)

דוגמה עם socket ו-JWT (ללא asyncio)

שימוש ב-socket סטנדרטי (מודול socket של Python) עם JWT — זרימה סינכרונית, מתאימה להבנת הבסיס ללא async:

קוד שרת — socket + JWT:

import socket, jwt, os, threading
from datetime import datetime, timedelta

SECRET_KEY = os.getenv("SECRET_KEY", "mysecret")
USERS_DB = { "alice": "1234", "bob": "5678" }

def create_token(username: str) -> str:
    payload = {
        "user_id": username,
        "exp": datetime.utcnow() + timedelta(hours=1)
    }
    return jwt.encode(payload, SECRET_KEY, algorithm="HS256")

def verify_token(token: str) -> dict | None:
    try:
        return jwt.decode(token, SECRET_KEY, algorithms=["HS256"])
    except jwt.ExpiredSignatureError:
        return None
    except jwt.InvalidTokenError:
        return None

def handle_client(conn, addr):
    print(f"חיבור מ-{addr}")
    with conn:
        # שלב 1: קבלת פרטי התחברות
        data = conn.recv(1024).decode()
        if ":" not in data:
            conn.sendall(b"שגיאה: פורמט שגוי")
            return
        username, password = data.split(":", 1)

        # שלב 2: אימות
        if USERS_DB.get(username) != password:
            conn.sendall("שגיאה: פרטים שגויים".encode())
            return

        # שלב 3: יצירת JWT ושליחה
        token = create_token(username)
        conn.sendall(token.encode())
        print(f"טוקן נשלח ל-{username}")

        # שלב 4: קבלת בקשה עם הטוקן
        token_data = conn.recv(4096).decode()
        payload = verify_token(token_data)
        if payload:
            conn.sendall(f"אומת בהצלחה: {payload['user_id']}".encode())
        else:
            conn.sendall(b"שגיאה: טוקן לא תקין")

def main():
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
    server.bind(("localhost", 9000))
    server.listen(5)
    print("שרת socket מאזין על פורט 9000...")
    while True:
        conn, addr = server.accept()
        thread = threading.Thread(target=handle_client, args=(conn, addr))
        thread.start()

main()

קוד לקוח — socket + JWT:

import socket

# הלקוח לא מכיר את SECRET_KEY — רק שולח credentials

def communicate(username: str, password: str):
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.connect(("localhost", 9000))

        # שלב 1: שליחת פרטי התחברות
        s.sendall(f"{username}:{password}".encode())

        # שלב 2: קבלת JWT מהשרת
        token = s.recv(4096).decode()
        if token.startswith("שגיאה"):
            print(f"כשל בהתחברות: {token}")
            return
        print(f"קיבלנו טוקן: {token[:40]}...")

        # שלב 3: שליחת הטוקן בבקשה הבאה
        s.sendall(token.encode())

        # שלב 4: קבלת תגובה
        response = s.recv(1024).decode()
        print(f"תגובה מהשרת: {response}")

communicate("alice", "1234")

סיכום זרימה — socket + JWT:

לקוח                                שרת
  |                                    |
  |── alice:1234 (TCP) ──────────────→ |  אימות מול USERS_DB
  |← JWT (bytes, תוקף שעה) ──────────|  jwt.encode() + SECRET_KEY
  |                                    |
  |── JWT (bytes) ───────────────────→ |  jwt.decode() + אימות
  |← אומת בהצלחה: alice ──────────── |
  |                                    |
  [ כל חיבור מטופל ב-Thread נפרד ]
  [ SECRET_KEY נמצא אך ורק בשרת   ]
הבדל מרכזי לעומת websockets:
socket — סינכרוני, דורש threading לריבוי לקוחות, שולח bytes ישירות
websockets — אסינכרוני (asyncio), מתאים לחיבורים מרובים בקלות
JWT עצמו זהה לחלוטין בשני המקרים

איך זה עובד?

  • הלקוח שולח credentials: username + password — לא יוצר טוקן בעצמו.
  • השרת מאמת ויוצר JWT: מכיל נתונים כמו מזהה משתמש ותפקיד, חתום דיגיטלית.
  • הלקוח מקבל ושומר את הטוקן: שולח אותו בכל בקשה הבאה.
  • השרת בודק את ה-JWT: מאמת שהחתימה תקינה. אם האימות נכשל — דוחה את הבקשה.

יתרונות וחסרונות

יתרונות חסרונות
החתימה מוודאת שהאסימון לא שונה רגיש לדליפה: אם נגנב, ניתן לשימוש
ניתן לשלוח בכותרת או כחלק מהבקשה אי אפשר לבטל אסימון בודד
מכיל את כל המידע הדרוש ללא תלות בשרת יש לנהל רשימת אסימונים מבוטלים לביטול

איך להעביר JWT?

JWT מועבר בדרך כלל כחלק מבקשות HTTP, בדרך כלל בכותרת האישור (Authorization Header):

Authorization: Bearer <JWT>

דוגמה נכונה להעברת JWT ב-WebSocket

⚠️ שגיאה נפוצה: הלקוח לא צריך להחזיק את ה-SECRET_KEY ולא ליצור טוקנים בעצמו!
הטוקן נוצר תמיד בשרת, לאחר אימות פרטי ההתחברות.

זרימה נכונה:

  • הלקוח שולח username + password לשרת
  • השרת מאמת ויוצר JWT (עם ה-SECRET_KEY שנמצא רק אצלו)
  • השרת שולח את הטוקן ללקוח
  • הלקוח משתמש בטוקן בבקשות הבאות — בלי לדעת את ה-SECRET_KEY

קוד שרת:

import asyncio, jwt, websockets, os
from datetime import datetime, timedelta

# SECRET_KEY נמצא רק בשרת — נטען ממשתני סביבה
SECRET_KEY = os.getenv("SECRET_KEY", "mysecret")

USERS_DB = { "alice": "password123" }

async def handle_connection(websocket):
    try:
        # שלב 1: קבלת פרטי התחברות מהלקוח
        credentials = await websocket.recv()  # "username:password"
        username, password = credentials.split(":")

        # שלב 2: אימות מול בסיס הנתונים
        if USERS_DB.get(username) != password:
            await websocket.send("שגיאה: פרטי התחברות שגויים")
            return

        # שלב 3: יצירת JWT בשרת ושליחה ללקוח
        payload = { "user_id": username, "exp": datetime.utcnow() + timedelta(hours=1) }
        token = jwt.encode(payload, SECRET_KEY, algorithm="HS256")
        await websocket.send(token)

        # שלב 4: אימות הטוקן בבקשה הבאה
        token_received = await websocket.recv()
        try:
            decoded = jwt.decode(token_received, SECRET_KEY, algorithms=["HS256"])
            await websocket.send(f"חיבור מאומת: {decoded['user_id']}")
        except jwt.ExpiredSignatureError:
            await websocket.send("שגיאה: JWT פג תוקף")
        except jwt.InvalidTokenError:
            await websocket.send("שגיאה: JWT לא תקין")
    except websockets.exceptions.ConnectionClosed:
        print("החיבור נסגר.")

async def main():
    async with websockets.serve(handle_connection, "localhost", 8765):
        print("שרת פועל...")
        await asyncio.Future()

asyncio.run(main())

import asyncio, websockets

# הלקוח לא מכיר את SECRET_KEY בכלל!

        await websocket.send(token)
        response = await websocket.recv()
        print(f"תגובה מהשרת: {response}")

asyncio.run(communicate())

שמירת SECRET_KEY — היכן ואיך?

כלל ברזל: לעולם לא שומרים את ה-SECRET_KEY ישירות בקוד!
הפתרון הנכון: משתני סביבה (Environment Variables)

יוצרים קובץ .env (שלא מועלה ל-Git) ומכניסים אליו את המפתח:

# .env  — לא מועלה ל-Git!
SECRET_KEY=my_super_secret_key_123

בקוד Python טוענים את המשתנה כך:

import os
from dotenv import load_dotenv

load_dotenv()
SECRET_KEY = os.getenv("SECRET_KEY")

האם משתפים את ה-SECRET_KEY עם הלקוח?

לא! זהו הלב של הרעיון. להלן ההבדל בין שני סוגי האלגוריתמים:

שרת לקוח
HS256 (סימטרי) יש לו את ה-secret אין לו
RS256 (אסימטרי) Private Key Public Key בלבד

זרימה נכונה עם HS256

1. הלקוח שולח: username + password
2. השרת בודק — ואז יוצר JWT ושולח ללקוח
3. הלקוח שומר את הטוקן ושולח אותו בכל בקשה
→ הלקוח מעולם לא יוצר טוקן בעצמו ומעולם לא רואה את ה-SECRET_KEY

הוספת הרשאות ב-JWT

ההרשאות נשמרות ב-Payload של הטוקן, בשדה permissions. להלן טבלת משתמשים לדוגמה:

משתמש תפקיד (role) הרשאות (permissions)
alice admin read, write, delete
bob viewer read בלבד

קוד שרת מלא עם הרשאות:

import asyncio, jwt, websockets, os
from datetime import datetime, timedelta

SECRET_KEY = os.getenv("SECRET_KEY", "mysecret")

USERS_DB = {
    "alice": {"password": "1234", "role": "admin",  "permissions": ["read","write","delete"]},
    "bob":   {"password": "5678", "role": "viewer", "permissions": ["read"]},
}

def create_token(username: str) -> str:
    user = USERS_DB[username]
    payload = {
        "user_id": username,
        "role": user["role"],
        "permissions": user["permissions"],  # רשימת הרשאות
        "exp": datetime.utcnow() + timedelta(hours=1)
    }
    return jwt.encode(payload, SECRET_KEY, algorithm="HS256")

def check_permission(payload: dict, required: str) -> bool:
    return required in payload.get("permissions", [])

async def handle_connection(websocket):
    login_data = await websocket.recv()  # "alice:1234"
    username, password = login_data.split(":")
    user = USERS_DB.get(username)
    if not user or user["password"] != password:
        await websocket.send("שגיאה: פרטי התחברות שגויים")
        return
    token = create_token(username)
    await websocket.send(token)

    while True:
        message = await websocket.recv()  # "TOKEN:ACTION"
        token_received, action = message.split(":", 1)
        try:
            payload = jwt.decode(token_received, SECRET_KEY, algorithms=["HS256"])
        except jwt.ExpiredSignatureError:
            await websocket.send("שגיאה: טוקן פג תוקף"); break
        except jwt.InvalidTokenError:
            await websocket.send("שגיאה: טוקן לא תקין"); break

        if action == "read" and check_permission(payload, "read"):
            await websocket.send(f"OK קריאת נתונים מותרת עבור {payload['user_id']}")
        elif action == "delete" and check_permission(payload, "delete"):
            await websocket.send(f"OK מחיקה מותרת עבור {payload['user_id']}")
        elif action == "delete":
            await websocket.send(f"DENIED אין הרשאת מחיקה ל-{payload['user_id']} (role: {payload['role']})")
        else:
            await websocket.send(f"DENIED פעולה לא מוכרת: {action}")

async def main():
    async with websockets.serve(handle_connection, "localhost", 8765):
        print("שרת פועל...")
        await asyncio.Future()

asyncio.run(main())

קוד לקוח עם הרשאות:

import asyncio, websockets

async def communicate(username: str, password: str):
    uri = "ws://localhost:8765"
    async with websockets.connect(uri) as websocket:
        await websocket.send(f"{username}:{password}")
        token = await websocket.recv()
        if token.startswith("שגיאה"):
            print(token); return
        print(f"התחברנו! קיבלנו טוקן.")
        for action in ["read", "delete"]:
            await websocket.send(f"{token}:{action}")
            response = await websocket.recv()
            print(f"בקשת {action}: {response}")

# alice — admin, יכולה הכל
asyncio.run(communicate("alice", "1234"))

# bob — viewer, רק קריאה
asyncio.run(communicate("bob", "5678"))

פלט צפוי:

# Alice (admin):
בקשת read:   OK קריאת נתונים מותרת עבור alice
בקשת delete: OK מחיקה מותרת עבור alice

# Bob (viewer):
בקשת read:   OK קריאת נתונים מותרת עבור bob
בקשת delete: DENIED אין הרשאת מחיקה ל-bob (role: viewer)

  |── JWT:delete ─────────────→ |  פענוח + בדיקת permissions
