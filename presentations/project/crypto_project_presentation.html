שלב מרכזי

שני צפנים מקוריים

לכל צופן נדרשים שלושה מרכיבים חובה

🔒 הצפנה
encrypt(plaintext, key) → ciphertext
🔓 פענוח
decrypt(ciphertext, key) → plaintext
💥 פריצה
crack(ciphertext) → plaintext (ללא מפתח!)
✅ אימות
הוכחה שהפריצה עובדת – דוגמה קוד ומסך
מה זה "מקורי"? שילוב יצירתי של עקרונות שנלמדו – לא פריצת דרך מתמטית. תנו לצופן שם ייחודי והסבירו את הרעיון.
  • החלפה (Substitution) – שינוי אות לאות אחרת לפי חוק קבוע
  • הזזה (Caesar / Affine) – שינוי אות לפי מפתח מספרי
  • ערבול (Transposition / Rail Fence) – שינוי סדר האותיות
  • מפתח משתנה (Vigenère) – מפתח שחוזר על עצמו
  • מפתח תלוי-הקשר – shift נגזר משם המשתמש / תאריך / מיקום
  • שילובים – לדוגמה: Rail Fence → Vigenère על הפלט
💡 שם דוגמאות: VigRail, RotatingAffine, MirrorVig, DateCaesar
📊 ניתוח תדירויות
מוצאים את האות הנפוצה ביותר בטקסט המוצפן → משווים לאות הנפוצה בשפה (בעברית: א, ה, ו, י). מתאים לצפני החלפה.
📝 תבניות מילים
מזהים מיפוי לאותיות על סמך מילון של תבניות מילים
🔨 Brute Force
ניסוי כל המפתחות האפשריים אחד אחד. מתאים למפתחות קצרים (כגון shift 1-25 ב-Caesar).
חשוב: בחרו את שיטת הפריצה המתאימה לצופן שלכם ונמקו את הבחירה בתיק הפרויקט.
אבטחה

מערכת הזדהות

כניסה, רישום ויציאה – בצורה מאובטחת

📝 Register
שם משתמש ייחודי + סיסמה (מינימום 6 תווים). נשמר ב-users.json כ-hash.
🔑 Login
אימות מול ה-hash השמור. עם הצלחה – המערכת נפתחת ומאפשרת הצפנה.
🚪 Logout
מנקה את סשן המשתמש הפעיל. כל פעולות ההצפנה חסומות שוב.
📄 פורמט users.json
{
  "users": {
    "alice": {
      "password_hash": "sha256:e3b0c4...",
      "created_at": "2025-01-01"
    }
  }
}
⛔ אסור לשמור סיסמה גלויה!
חובה: hashlib.sha256(pwd.encode()).hexdigest()
הפרה תגרור הפחתת ניקוד משמעותית.

בדיקות

בדיקות pytest

לפחות 10 בדיקות עצמאיות – כולן חייבות לעבור

בדיקות חובה:
  • Round-trip לכל אחד מ-2 הצפנים
  • פענוח עם מפתח שגוי לא מחזיר מקור
  • Register שומר משתמש ב-JSON
  • Login תקין מצליח (True)
  • Login שגוי נכשל (False)
  • שם משתמש כפול מחזיר שגיאה
  • פונקציית crack מצליחה לפחות פעם אחת
  • קלט ריק לא קורס את הקוד
דוגמה:
def test_cipher1_round_trip():
    from ciphers.cipher1 import (
        encrypt, decrypt
    )
    plaintext = "HELLO"
    key = 3
    encrypted = encrypt(plaintext, key)
    result   = decrypt(encrypted, key)
    assert result == plaintext
הרצה: pytest tests/ -v
