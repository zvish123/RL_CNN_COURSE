# Slide 6: ההיגיון מאחורי הקלעים 🔬

## איך המערכת מזהה?

### דוגמה: Gaming vs Streaming

| תכונה | Gaming 🎮 | Streaming 📹 |
|-------|-----------|--------------|
| גודל Packet | קטן (100B) | גדול (1400B) |
| תדירות | גבוהה (64/sec) | בינונית (30/sec) |
| כיווניות | סימטרי ⇆ | אסימטרי ⇓ |
| קביעות | קבוע מאוד | Bursts |
