app.launch(theme=gr.themes.Soft())

🎨 מה זה CSS ואיך משתמשים בו כאן? ▼

CSS (Cascading Style Sheets) הוא שפת עיצוב שאומרת לדפדפן איך להציג אלמנטים – צבע, גודל, כיוון, ריווח ועוד. ב-Gradio אפשר להזריק CSS חופשי דרך פרמטר css של gr.Blocks.

🔎 מבנה כלל CSS
selector {
    property: value;
}

selector – על מה להחיל (תג, מחלקה, id)
property – מה לשנות
value – לאיזה ערך

📌 שלושת הכללים בקוד שלנו
  • * { direction: rtl; }
    כל האלמנטים – טקסט מימין לשמאל (עברית)
  • .slider-ltr * { direction: ltr !important; }
    הסליידר עצמו – ירוץ משמאל לימין (כרגיל)
  • .slider-ltr label { text-align: right; }
    אבל התווית מעל הסליידר – תישאר עברית ימינה
⚡ !important – מה זה?

Gradio מוסיף CSS משל עצמו לכל רכיב. !important אומר "הכלל שלי מנצח את כולם" – גם אם Gradio קבע משהו אחר.

השתמשו בו רק כשצריך לדרוס עיצוב של ספרייה חיצונית.

🏷 מחלקה (class) ב-Gradio

כשכותבים elem_classes=["slider-ltr"] ב-Gradio, הרכיב מקבל את המחלקה slider-ltr ב-HTML. ואז ה-CSS שכתבנו עם .slider-ltr מחיל עליו את העיצוב.

def caesar_cipher(text: str, shift: int, mode: str) -> str:
    # הצפן = הזזה קדימה, פיענוח = הזזה אחורה
    if mode == "פענח":
        shift = -shift
    result = []
    for ch in text:
        if ch.isalpha():
            base = ord('A') if ch.isupper() else ord('a')
            result.append(chr(((ord(ch) - base + shift) % 26) + base))
        else:
            result.append(ch)
    return ''.join(result)

# בדיקה בטרמינל – לפני Gradio!
print(caesar_cipher("Hello", 3, "הצפן"))   # → Khoor
print(caesar_cipher("Khoor", 3, "פענח"))   # → Hello

✏️ כתוב ובדוק בעצמך

כתבו את rail_fence_cipher(text, rails, mode):

  • צרו rails רשימות ריקות
  • עברו על האותיות – כל אות הולכת ל-rail שמספרו 0, 1, 2, …, rails-1, rails-2, …, 1, 0, 1, …
  • לפיענוח: חשבו אילו אינדקסים שייכים לכל rail, ואז קראו לפי הסדר המקורי

בדקו בטרמינל: rail_fence_cipher("HELLOWORLD", 3, "הצפן") → HOLELWRDLO

gr.Textbox

תיבת טקסט – לכתיבת קלט או הצגת פלט

קלט / פלט
🎯 מטרה

הרכיב הנפוץ ביותר ב-Gradio. משמש גם כקלט (המשתמש כותב בו) וגם כפלט (הקוד מחזיר לתוכו תוצאה).

כשמשתמשים בו כפלט – הוסיפו interactive=False כדי שהמשתמש לא יוכל לערוך.

🔐 לצפן: קלט לטקסט המקורי + פלט לטקסט המוצפן
⚙️ פרמטרים חשובים
  • label – כותרת מעל השדה
  • lines=4 – גובה בשורות (1 = שורה אחת)
  • placeholder – טקסט אפור רמז
  • value – ערך ברירת מחדל
  • type="password" – מסתיר את הטקסט (לסיסמאות)
  • interactive=False – רק לקריאה (לפלט)
# כקלט – המשתמש מזין טקסט
txt_input = gr.Textbox(
    label="טקסט להצפנה",
    lines=4,
    placeholder="הקלד כאן את הטקסט שלך..."
)

# כפלט – הקוד כותב לתוכו (לא ניתן לעריכה)
txt_output = gr.Textbox(
    label="תוצאת ההצפנה",
    lines=4,
    interactive=False
)

gr.Slider

מחוון לבחירת ערך מספרי בטווח נתון

קלט מספרי
🎯 מטרה

מאפשר בחירת מספר ע"י גרירה. מושלם כשיש טווח ידוע ורוצים שהמשתמש לא יכניס ערך לא חוקי.

עדיף על gr.Number כשהטווח מוגבל – לא צריך validation ידני!

🔐 לצפן קיסר: מפתח 1–25 | לזיגזג: מספר פסים 2–8

key_slider.change(fn=caesar_cipher, inputs=[...], outputs=[...])

gr.Radio

בחירת אפשרות אחת מתוך רשימה – כפתורי רדיו

בחירה בלעדית
🎯 מטרה

כשיש 2–5 אפשרויות ורוצים שהמשתמש יבחר בדיוק אחת. כל האפשרויות נראות בבת אחת.

ההבדל מ-Dropdown: Radio מראה הכל, Dropdown מסתיר עד לחיצה. עדיף Radio עד ~5 אפשרויות.

  • value – בחירת ברירת מחדל
  • label – כותרת
mode = gr.Radio(

    value="הצפן",
    label="בחר פעולה"

gr.Number

gr.Markdown

הצגת טקסט עשיר – כותרות, הדגשות, קוד

תצוגת תוכן
🎯 מטרה

להצגת הסברים, כותרות, ותוצאות מעוצבות. תומך בסינטקס Markdown מלא: **מודגש**, # כותרת, רשימות, קוד מוטבע.

אפשר גם לעדכן Markdown דינמית מהפונקציה!

🔐 הצגת טבלת ההחלפה, ניתוח תדירות בטקסט
⚙️ שימוש כפלט דינמי
  • הגדירו כרכיב ב-outputs
  • הפונקציה מחזירה מחרוזת Markdown
  • Gradio מרנדר אותה אוטומטית

gr.Button + .click()

כפתור לחיצה – מפעיל פונקציה Python

אירוע לחיצה
🎯 מטרה

הגשר בין ה-UI לקוד Python. כשהמשתמש לוחץ, Gradio אוסף את ערכי ה-inputs ומעביר לפונקציה, ואת התוצאה שם ב-outputs.

🔐 כפתור "הצפן", "פענח", "פרוץ", "ייצר מפתח"
⚙️ פרמטרים של .click()
  • fn – הפונקציה להפעיל
  • inputs=[...] – רשימת רכיבי קלט
  • outputs=[...] – רשימת רכיבי פלט
  • variant="primary" – כפתור ראשי (צבעוני)
  • variant="stop" – כפתור עצור (אדום)
btn = gr.Button("🔐 הצפן עכשיו", variant="primary")
btn_clear = gr.Button("🗑 נקה")

# קישור לפונקציה
btn.click(
    fn=caesar_cipher,
    inputs=[txt_in, key_slider, mode_radio],
    outputs=[txt_out]
)

# כפתור ניקוי – הפונקציה מחזירה "" לכל השדות
btn_clear.click(fn=lambda: ("", ""), outputs=[txt_in, txt_out])
