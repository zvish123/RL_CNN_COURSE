02 עקרונות SOLID — הגדרה ויישום בפרויקט
S
Single Responsibility — אחריות יחידה כל מחלקה אחראית על דבר אחד בלבד. שינוי בלוגיקת register לא אמור לגעת בקוד של broadcast.
ChatServer = מתאם בלבד ClientManager = חיבורים UserRepository = משתמשים RegisterHandler = לוגיקת register
O
Open/Closed — פתוח להרחבה, סגור לשינוי רוצים להוסיף פקודה list_users? כותבים מחלקה חדשה. handle_client לא נגע. server.py לא נגע. אפס elif חדש.
הוספת PrivateMsgHandler → קובץ חדש בלבד ChatServer ← ללא שינוי ✓
L
Liskov Substitution — החלפת תת-מחלקה כל handler שיורש מ-BaseCommandHandler חייב לעבוד זהה. InMemoryUserRepo ו-SQLUserRepo — שניהם עובדים כ-IUserRepository. השרת לא יודע ולא אכפת לו.
RegisterHandler extends BaseCommandHandler ✓ SQLUserRepo extends IUserRepository ✓
I
Interface Segregation — ממשקים ממוקדים IUserRepository חושפת רק 3 מתודות — register_user, validate_user, user_exists. לא 30 מתודות שרוב ה-handlers לא צריכים. BaseCommandHandler חושפת רק 2.
IUserRepository: 3 מתודות BaseCommandHandler: can_handle + handle
D
Dependency Inversion — תלות בהפשטות ChatServer מקבל IUserRepository — לא InMemoryUserRepository. רוצה SQLite? משנים שורה אחת. ChatServer, handlers, ClientManager — לא נגעים.
CommandContext(user_repo: IUserRepository, client_manager)
SOLID בפייתון: Python לא מאכף interfaces כמו Java. ABCs נותנות הגנה חלקית — אם שכחת לממש @abstractmethod, Python זורק שגיאה בעת יצירת instance. SOLID הוא מדריך תכנון, לא חוק נוקשה.

BaseCommandHandler — ABC
from abc import ABC, abstractmethod class BaseCommandHandler(ABC): @abstractmethod def can_handle(self, action: str) -> bool: pass @abstractmethod def handle(self, msg, socket, protocol, context ) -> ProtocolMessage: pass # handle() מחזיר — לא שולח! # השרת שולח. הפרדת אחריות.
RegisterCommandHandler
class RegisterCommandHandler(BaseCommandHandler): def can_handle(self, action) -> bool: return action == "register" def handle(self, msg, sock, proto, ctx): success = ctx.user_repo.register_user( msg.username, msg.password) if not success: return ProtocolMessage( action="register", status="error", error="User already exists") return ProtocolMessage( action="register", status="ok", message="Registered successfully")
IUserRepository — הפשטה
class IUserRepository(ABC): @abstractmethod def register_user(self, username, password) -> bool: pass @abstractmethod def validate_user(self, username, password) -> bool: pass @abstractmethod def user_exists(self, username) -> bool: pass # עכשיו: InMemoryUserRepository # עתיד: SQLUserRepository (שורה אחת!)
