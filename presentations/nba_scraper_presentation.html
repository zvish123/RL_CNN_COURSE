איתור אלמנטים 🎯

1

איך למצוא את הטבלה?

1. לחיצה ימנית על הטבלה באתר

2. בחירה ב"Inspect Element"

3. חיפוש ה-ID או CLASS

מה נמצא באתר NBA:

<table id="per_game_stats">
2

המתנה חכמה

# לא טוב: time.sleep(5)  # חיכוי עיוור
# טוב:
wait = WebDriverWait(driver, 10)
table = wait.until(EC.presence_of_element_located((By.ID, "per_game_stats")))

טיפ חשוב: תמיד חכו שהאלמנט יטען! אתרים מודרניים איטיים

קריאת הנתונים 📖

1

קודם הכותרות

headers = []
header_row = table.find_element(By.TAG_NAME, "thead")
for th in header_row.find_elements(By.TAG_NAME, "th"):
    headers.append(th.text.strip())
print("כותרות:", headers)
2

אחר כך הנתונים

players_data = []
tbody = table.find_element(By.TAG_NAME, "tbody")
rows = tbody.find_elements(By.TAG_NAME, "tr")

for row in rows:
    cells = row.find_elements(By.TAG_NAME, "td")
    player_info = {}
    
    for i, cell in enumerate(cells):
        if i < len(headers):
            player_info[headers[i]] = cell.text.strip()
    
    players_data.append(player_info)

מה קורה כאן? עוברים שורה אחר שורה, תא אחר תא, ובונים רשימת שחקנים

def convert_to_number(value):
    if value and value != "":
        try:
            if "." in value:
                return float(value)
            elif value.isdigit():
                return int(value)
        except:
            pass
    return value

בעיות נפוצות ופתרונות 🔧

בעיה #1: "chromedriver not found"

פתרון: ודאו שהדרייבר בתיקייה או ב-PATH

בעיה #2: "Element not found"

פתרון: האתר השתנה - עדכנו את הסלקטורים

בעיה #3: "TimeoutException"

פתרון: האינטרנט איטי - הגדילו timeout

עצות זהב:

  • 🧪 בדקו על דף קטן לפני האתר הגדול
  • 🖨️ הדפיסו מידע לבדיקה: print(cell.text)
  • 🔍 השתמשו ב-browser developer tools
  • ⏰ אל תמהרו - תנו זמן לאתר להיטען
