🎓 סיכום וטיפים

✅ מה למדנו?

  • CREATE - יצירת טבלאות ומבני נתונים
  • INSERT - הוספת נתונים חדשים
  • SELECT - שליפת נתונים (פשוט ומורכב)
  • UPDATE - עדכון נתונים קיימים
  • DELETE - מחיקת נתונים
  • JOIN - חיבור טבלאות
  • GROUP BY & HAVING - קיבוץ ואגרגציה
  • ORDER BY - מיון תוצאות

⚠️ טעויות נפוצות להימנע מהן:

  • שכחת WHERE ב-UPDATE או DELETE (יעדכן/ימחק הכל!)
  • שכחת מרכאות בודדות (') סביב טקסט
  • שימוש במפתח ראשי כפול
  • ניסיון למחוק/לעדכן רשומה עם foreign key constraints
  • שכחת לציין את העמודות ב-INSERT
  • טעויות כתיב בשמות טבלאות ועמודות

💡 טיפים לכתיבת SQL טוב:

  • תמיד בדקו עם SELECT לפני UPDATE/DELETE
  • השתמשו בשמות ברורים לטבלאות ועמודות
  • הוסיפו הערות (--) לקוד המורכב
  • השתמשו ב-aliases (כינויים) לטבלאות בשאילתות מורכבות
  • תרגלו על נתונים מדומים לפני עבודה על נתונים אמיתיים
  • שמרו גיבויים לפני שינויים גדולים!
