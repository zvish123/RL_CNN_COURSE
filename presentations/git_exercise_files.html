🛠️ פקודות Git נפוצות לתרגיל

פקודות שהתלמידים ישתמשו במהלך התרגיל
⚡ פקודות בסיסיות 
# שכפול הפרויקט
git clone [URL]

# בדיקת סטטוס
git status

# מעבר בין branches
git checkout [branch-name]
git checkout -b [new-branch-name]

# הוספת קבצים לcommit
git add [filename]
git add .

# יצירת commit
git commit -m "הודעת commit"

# דחיפה לשרת
git push origin [branch-name]

# עדכון מהשרת
git pull origin master
🔥 פקודות לתיקוני חירום 
# יצירת hotfix branch
git checkout master
git pull origin master
git checkout -b hotfix1

# אחרי תיקון הקובץ
git add index.html
git commit -m "HOTFIX: Fix quote author"
git push origin hotfix1

# פתרון merge conflict
git merge master
# עריכת הקובץ לפתרון הconflict
git add index.html
git commit -m "Resolve merge conflict"
