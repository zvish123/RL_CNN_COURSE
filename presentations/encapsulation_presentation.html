🔒 Encapsulation בתכנות מונחה עצמים

מה זה Encapsulation?

עקרון המאפשר להסתיר פרטי מימוש פנימיים של מחלקה ולחשוף רק ממשק ציבורי מוגדר

דמיון מהחיים:

כמו רכב - אתה יכול להשתמש בהגה, דוושות וכפתורים, אבל אין לך גישה ישירה למנוע הפנימי!

🎯 המטרה: שמירה על שלמות הנתונים וביטחון הקוד

רמות הגישה בפייתון

🔓 Public (ציבורי)

class Person:
    def __init__(self, name):
        self.name = name  # תכונה ציבורית
    
    def greet(self):  # מתודה ציבורית
        return f"שלום, אני {self.name}"

person = Person("יוסי")
print(person.name)  # גישה ישירה מותרת

🔒 Private (פרטי)

class BankAccount:
    def __init__(self, balance):
        self.__balance = balance  # תכונה פרטית (__)
    
    def __calculate_interest(self):  # מתודה פרטית
        return self.__balance * 0.05

account = BankAccount(1000)
# print(account.__balance)  # ❌ שגיאה! לא ניתן לגשת ישירות

Protected Access (גישה מוגנת)

🔐 Protected (_single_underscore)

class Animal:
    def __init__(self, name):
        self._name = name  # מוגן - לשימוש פנימי או בירושה
    
    def _internal_method(self):
        return f"פעולה פנימית של {self._name}"

class Dog(Animal):
    def bark(self):
        return f"{self._name} נובח!"  # גישה מוגן מותרת בירושה

dog = Dog("רקס")
print(dog._name)  # עובד, אבל לא מומלץ!

💡 הערה חשובה: Protected הוא רק קונבנציה - פייתון לא אוכף אותו באופן קשיח!

Getters ו-Setters עם Property

class Temperature:
    def __init__(self, celsius):
        self.__celsius = celsius
    
    @property
    def celsius(self):  # Getter
        return self.__celsius
    
    @celsius.setter
    def celsius(self, value):  # Setter עם וולידציה
        if value < -273.15:
            raise ValueError("טמפרטורה לא יכולה להיות נמוכה מ-273.15°C")
        self.__celsius = value
    
    @property
    def fahrenheit(self):
        return (self.__celsius * 9/5) + 32

# שימוש:
temp = Temperature(25)
print(temp.celsius)     # 25
print(temp.fahrenheit)  # 77.0

temp.celsius = 30       # שימוש ב-setter
# temp.celsius = -300   # ❌ יזרוק שגיאה!

דוגמה מעשית: מחלקת Student

class Student:
    def __init__(self, name, student_id):
        self.name = name                    # ציבורי
        self._grade = None                  # מוגן
        self.__student_id = student_id      # פרטי
        self.__grades = []                  # פרטי
    
    def add_grade(self, grade):
        if 0 <= grade <= 100:
            self.__grades.append(grade)
        else:
            raise ValueError("ציון חייב להיות בין 0 ל-100")
    
    @property
    def average_grade(self):
        if not self.__grades:
            return 0
        return sum(self.__grades) / len(self.__grades)
    
    def __str__(self):
        return f"תלמיד: {self.name}, ממוצע: {self.average_grade:.1f}"

# שימוש:
student = Student("שרה כהן", "123456")
student.add_grade(85)
student.add_grade(92)
print(student)                              # תלמיד: שרה כהן, ממוצע: 88.5

# print(student.__student_id)               # ❌ לא עובד
# print(student._Student__student_id)       # ✅ Name mangling - עובד אבל לא מומלץ!

מגבלות ה-Encapsulation בפייתון

❌ מגבלות:

  • אין Private אמיתי - רק Name Mangling
  • ניתן לגשת דרך _ClassName__attribute
  • Protected הוא רק קונבנציה
  • הכל נגיש בזמן ריצה

✅ יתרונות:

  • גמישות וחופש למפתח
  • אפשרות לדיבוג קל יותר
  • פייתון סומך על המפתח
  • "We're all consenting adults"

דוגמה למגבלה:

class Secret:
    def __init__(self):
        self.__secret = "סוד גדול!"

s = Secret()
print(s._Secret__secret)  # "סוד גדול!" - עובד!

# פייתון משנה את השם ל: _ClassName__attributeName
print(dir(s))  # תראה '_Secret__secret' ברשימה

🎯 Best Practices

✅ מה כדאי לעשות:

  • השתמש ב-@property לגישה מבוקרת לנתונים
  • הגדר תכונות כ-Private (__) אם הן פנימיות
  • השתמש ב-Protected (_) לתכונות לירושה
  • הוסף validation ב-Setters
  • כתב תיעוד ברור לממשק הציבורי

❌ מה לא כדאי לעשות:

  • לא לגשת לתכונות פרטיות דרך Name Mangling
  • לא להפוך הכל לפרטי ללא סיבה
  • לא לשכוח validation בעת שינוי נתונים
  • לא להשתמש ב-Protected בלי צורך אמיתי

📝 סיכום

העקרונות המרכזיים:

🔓 Public

גישה חופשית מכל מקום

self.attribute

🔐 Protected

לשימוש פנימי וירושה

self._attribute

🔒 Private

Name Mangling, לא גישה ישירה

self.__attribute
