🏛️ הצפנת קיסר - הרקע ההיסטורי

יוליוס קיסר השתמש בשיטה פשוטה אך גאונית:
הזיז כל אות באלפבית במספר מקומות קבוע!

העיקרון: כל אות מוחלפת באות אחרת על פי היסט קבוע
A → D (היסט 3)
B → E (היסט 3)
C → F (היסט 3)

⚙️ איך ההצפנה עובדה?

שלושה צעדים פשוטים:

1️⃣ בוחרים מספר היסט (המפתח)
2️⃣ כל אות במסר מוזזת לפי ההיסט
3️⃣ אם עוברים את Z, חוזרים ל-A
💡 הנוסחה המתמטית:
אות מוצפנת = (אות מקורית + היסט) % 26

🔤 דוגמה: הצפנת המילה HELLO

מסר מקורי:

HELLO

מסר מוצפן (היסט 3):

KHOOR
הסבר:
H (7) + 3 = K (10)
E (4) + 3 = H (7)
L (11) + 3 = O (14)
L (11) + 3 = O (14)
O (14) + 3 = R (17)

🐍 עבודה עם מחרוזות בפייתון

פונקציות חשובות:

text = "Hello World"

# המרה לאותיות גדולות
text.upper() # "HELLO WORLD"

# המרה לאותיות קטנות
text.lower() # "hello world"

# חיפוש מיקום אות
text.find("o") # 4

# בדיקת תוכן
"H" in text # True

✅ פונקציות בדיקה למחרוזות

isalpha() - בודק אם כל התווים הם אותיות
"Hello".isalpha() → True
isdigit() - בודק אם כל התווים הם ספרות
"123".isdigit() → True
isalnum() - בודק אם כל התווים הם אותיות או ספרות
"Hello123".isalnum() → True
isupper() / islower() - בודק רישיות
"HELLO".isupper() → True

💻 מימוש פונקציית ההצפנה

def encrypt(key, message):
message = message.upper()
ABC = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
result = ""

for letter in message:
if letter in ABC:
index = (ABC.find(letter) + key) % 26
result = result + ABC[index]
else:
result = result + letter

return result
שימו לב: התו % מבצע פעולת מודולו - מחזיר את השארית של חלוקה

🔓 מימוש פונקציית הפענוח

def decrypt(key, message):
message = message.upper()
ABC = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
result = ""

for letter in message:
if letter in ABC:
index = (ABC.find(letter) - key) % 26
result = result + ABC[index]
else:
result = result + letter

return result
ההבדל היחיד: במקום חיבור (+) משתמשים בחיסור (-)
